Overview of CSS Borders and Border Radius

CSS provides properties to style borders around elements. You can customize the style, color, and width of borders, as well as add rounded edges using the border-radius property.

Border Style and Width

Border styles include solid, dashed, dotted, and double, among others. You can also specify the border’s width and color. Here are some examples:


h2 {
    background-color: aqua;
    border-style: solid;       /* Defines a solid border */
    border-width: 2px;         /* Sets the border width to 2px */
    border-color: black;       /* Sets the border color to black */
    /* This can also be simplified as: */
    border: solid 2px black;
}

Applying Borders to Different Elements

Borders can be applied to paragraphs, articles, or any element. Here, a dashed red border is added to paragraphs:


p {
    border: dashed 2px red;    /* Adds a 2px dashed red border */
}

Border Radius for Rounded Corners

The border-radius property is used to create rounded corners. You can apply it uniformly or target specific corners:


article {
    background-color: antiquewhite;
    border: solid green 5px;      /* Solid green border with 5px width */
    border-radius: 40px;          /* Rounds all corners to 40px */
}

#anyonecurve {
    background-color: blueviolet;
    border: solid 2px black;
    border-top-left-radius: 20px;     /* Rounds the top-left corner */
    border-bottom-right-radius: 20px; /* Rounds the bottom-right corner */
}

This section demonstrates how to apply borders and rounded corners to various elements, allowing for creative styling and emphasis.

Overview of CSS Display Property

The display property in CSS determines how an element behaves in the layout of the page. Common values include block, inline, inline-block, and flex.

Example 1: Inline Element without Display Modification

Since span is an inline element, its width and height won’t increase as expected. The following example shows an inline span element:


#withoutDisplayFeatureInCss {
    width: 200px;
    height: 300px;
    background-color: green;
}

Result: The span element retains its inline behavior and does not expand in size.

Example 2: Using display: inline-block for Sizing Control

Setting display: inline-block on an inline element like span allows it to accept width and height properties.


#withDisplayFeatureInCss {
    display: inline-block;
    width: 300px;
    height: 300px;
    background-color: green;
}

Result: The span element now expands to the specified size.

Example 3: Using display: flex for Flexible Layouts

The display: flex property makes the element a flex container, allowing flexible arrangement of child elements.


#withDisplayFlexFeatureInCss {
    display: flex;
    width: 300px;
    height: 300px;
    background-color: green;
}

Result: The container becomes flexible, which is helpful for responsive layouts and aligns child elements in rows or columns as needed.

This section demonstrates different uses of the display property to manage layout behavior, especially for inline elements like span.

CSS Box Model

Overview of the CSS Box Model:

The CSS Box Model represents the structure of an HTML element as a rectangular box. It includes four parts: content, padding, border, and margin. Each part adds space around the element in different ways.

Parts of the Box Model

  • Content: The actual content of the element, such as text or images.
  • Padding: Space around the content, inside the border.
  • Border: A border around the padding and content.
  • Margin: Space outside the border, separating the element from others.

Example: Padding and Margin in Action

Padding: Adds space within the element, between the content and the border.

Margin: Adds space outside the element, separating it from other elements.


/* Example of Padding */
.element {
    padding: 20px 30px; /* 20px for top and bottom, 30px for left and right */
    background-color: lightblue;
}

/* Example of Margin */
.element {
    margin: 10px; /* 10px space outside the element on all sides */
    background-color: lightcoral;
}

Result: In this example, the padding creates space within the element, while the margin creates space outside it. This layout can be visualized using browser inspection tools.

The Box Model is foundational for understanding layout in CSS, and using padding, borders, and margins can enhance element spacing and organization.
