What Is Bootstrap

Introduction to Bootstrap

Bootstrap is a powerful, feature-packed frontend toolkit that helps you quickly build responsive, modern web pages. You can include Bootstrap by either downloading the files or using a CDN link for the CSS and JavaScript.

CDN Links:


<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Bootstrap JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Container

Containers are used to center and pad the content. In Bootstrap, they are the main layout element that enables responsive design. 


<div class="container">
    <h1>Responsive Container</h1>
</div>

Badge Component

Badges are used for notifications or counters and are commonly added next to text or inside headings. 


<h1>Messages <span class="badge bg-secondary">4</span></h1>

Alert Component

The Alert component provides feedback messages for different user actions, such as success, warnings, or errors. 


<div class="alert alert-success" role="alert">
    This is a success alert!
</div>

Navbar Component

Bootstrap's Navbar component allows you to create responsive navigation bars with options to collapse on smaller screens. 


<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <a class="navbar-brand" href="#">Navbar</a>
    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
        <ul class="navbar-nav">
            <li class="nav-item active">
                <a class="nav-link" href="#">Home</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Features</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Pricing</a>
            </li>
        </ul>
    </div>
</nav>

Grid System

Bootstrap’s grid system is based on a 12-column layout and is designed to create responsive layouts. Each row is divided into columns with classes like col-*. 


<div class="container">
    <div class="row">
        <div class="col-6">Column 1</div>
        <div class="col-6">Column 2</div>
    </div>
</div>

Forms Component

Bootstrap provides form components with consistent styling, including controls for text input, checkboxes, radio buttons, and more. 


<form>
    <div class="form-group">
        <label for="exampleInputEmail1">Email address</label>
        <input type="email" class="form-control" id="exampleInputEmail1">
    </div>
    <div class="form-check">
        <input type="checkbox" class="form-check-input" id="exampleCheck1">
        <label class="form-check-label" for="exampleCheck1">Check me out</label>
    </div>
</form>

Bootstrap Button Examples

Introduction to Bootstrap Buttons

Bootstrap provides various button styles that enhance the appearance and usability of your buttons with minimal effort. You can create buttons using Bootstrap's predefined classes to ensure consistency and responsiveness across devices.

Example of Bootstrap Buttons


<button>Click Me!</button>  <!-- By me -->
<button class="btn">By Bootstrap Button</button>  <!-- Styles are given by Bootstrap -->

<button class="btn btn-danger">Click Me!</button>
<button class="btn btn-success">Click Me!</button>

Including Bootstrap in Your Project

To use Bootstrap, include the CSS and JavaScript files either by downloading them or using a CDN. Here’s how to include them in your HTML file: 


<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Custom Styling

If you want to modify the default Bootstrap styles or add your custom styles, you can create a separate CSS file and link it in your HTML. Here’s an example: 


<link rel="stylesheet" href="style.css">  <!-- Use this for custom styles -->
