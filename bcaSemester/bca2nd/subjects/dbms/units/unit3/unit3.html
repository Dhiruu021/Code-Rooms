Relational Data Model Concepts

Introduction to the relational data model, its components, and how it represents data in the form of tables (relations).

Integrity Constraints

Learn about integrity constraints that ensure the accuracy and consistency of data in a relational database.

Entity Integrity

Explore entity integrity constraints that ensure each row (tuple) in a relation is uniquely identifiable.

Referential Integrity

Understand referential integrity constraints that ensure relationships between tables are maintained accurately.

Keys Constraints

Learn about keys in relational databases, including primary keys, foreign keys, and unique keys, and their role in enforcing constraints.

Domain Constraints

Explore domain constraints that specify valid values for attributes in a relation.

Relational Algebra

Introduction to relational algebra, a procedural query language for relational databases that uses operators to manipulate relations.

Relational Calculus

Explore relational calculus, a declarative query language used to express queries without describing the procedure to be followed.

Tuple Calculus

Learn about tuple calculus, a type of relational calculus that focuses on specifying the tuples to be selected based on certain conditions.

Domain Calculus

Understand domain calculus, which focuses on specifying the domain (set of values) that the attributes should satisfy for the tuples to be selected.
