Embedding Media and External Content

Images

The <img> tag is used to embed images, with options for loading control using the loading attribute.


<img src="/basics/06_akhand.jpeg" alt="Example Image" width="100px" loading="eager">
<img src="/basics/06_akhand.jpeg" alt="Example Image" width="100px" loading="lazy">
  • loading="eager": Loads the image immediately.
  • loading="lazy": Defers image loading until it is in view.

Audio

The <audio> tag embeds audio files. The controls attribute allows users to play or pause audio.


<audio src="audio-file.mp3" controls></audio>

<audio controls>
   <source src="another-audio.mp3" type="audio/mp3">
</audio>

Video

The <video> tag is used to embed video files on a web page.


<video src="/video/example.mp4" height="150px"></video>
<video src="/video/example.mp4" height="250px" controls></video>

<video controls height="250px">
   <source src="/video/example.mp4" type="video/mp4">
</video>
  • controls: Shows playback controls (play, pause, etc.).

Horizontal Rule

The <hr> tag is used to create a horizontal line, typically for section separation.


<hr>

Iframes

The <iframe> tag embeds external web pages or content within a web page. Useful for embedding websites, videos, or documents.


<iframe src="/project_1/index.html" frameborder="1" title="Portfolio" height="300px"></iframe>

<iframe src="https://www.youtube.com/embed/k7ELO356Npo?autoplay=1&mute=1" frameborder="1" title="YouTube Video" height="300px"></iframe>

<iframe src="https://www.wikipedia.org/" frameborder="1" title="Wikipedia" height="300px"></iframe>
  • src: URL of the embedded content.
  • frameborder: Width of the iframe's border.
  • autoplay & mute (YouTube): Controls playback options for embedded YouTube videos.
