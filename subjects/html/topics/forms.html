HTML Forms

Basic Form


<form>  <!-- Start of the form -->
    <label for="email">Enter Email ID:</label>  <!-- Label for email input -->
    <input type="email" id="email" placeholder="Enter Your Email.">  <!-- Email input field -->
    <br><br>
    
    <label for="password">Enter Password:</label>  <!-- Label for password input -->
    <input type="password" id="password">  <!-- Password input field -->
    <br><br>

    <input type="submit" value="SUBMIT HERE.">  <!-- Submit button -->
</form>  <!-- End of the form -->

<h3>Basic Form completed.</h3>

HTML Advanced Input Types

Basic Text and Button Inputs

These are common form elements used for entering text and clicking buttons.


<form>
    <!-- Text Input for simple text fields -->
    <label for="textfield">TextField:</label>
    <input type="text" name="textfield" id="textfield">

    <!-- Button for clickable actions without form submission -->
    <label for="button">Button:</label>
    <input type="button" name="button" id="button" value="UnSubscribe">

    <!-- Submit Button to submit form data to a server -->
    <input type="submit" name="button" id="button" value="UnSubscribe">
</form>

Radio Buttons

Radio buttons let users select only one option from a set. Each option should have the same name attribute.


<!-- A single radio button (non-selectable alone) -->
<label for="Radiobutton">Creating a Radio Button:</label>
<input type="radio" name="radio" id="Radiobutton">

<!-- Grouped Radio Buttons for gender selection -->
<label for="radio1">Male:</label>
<input type="radio" name="radio" id="radio1">

<label for="radio2">Female:</label>
<input type="radio" name="radio" id="radio2">

<label for="radio3">Transgender:</label>
<input type="radio" name="radio" id="radio3">

Checkboxes

Checkboxes allow users to select multiple options independently. They can share the same name or have different names, depending on the desired behavior.


<!-- Single Checkbox -->
<label for="checkbox1">Checkbox Button:</label>
<input type="checkbox" name="checkbox1" id="checkbox1">

<!-- Multiple Checkboxes for selecting languages -->
<label for="checkbox1">English:</label>
<input type="checkbox" name="checkbox" id="checkbox1">

<label for="checkbox2">Hindi:</label>
<input type="checkbox" name="checkbox" id="checkbox2">

HTML Forms Practices:
Event Registration and Job Application

Event Registration Form

This form allows users to register for an event, collecting details like name, email, phone number, occupation, interests, and additional comments.


<!-- Event Registration Form -->
<form>
    <h1>Event Registration</h1>
    
    <!-- Name Field -->
    <label for="naam">Name:</label>
    <input type="text" name="naam" id="naam">
    
    <!-- Email Field -->
    <label for="email">Email Id:</label>
    <input type="email" name="email" id="email">
    
    <!-- Phone Number with Pattern -->
    <label for="tel">Phone No.:</label>
    <small>Format: 123-456-7890</small>
    <input type="tel" name="telephone" id="tel" pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}" required>
    
    <!-- Occupation Selection -->
    <label for="occupation">Occupation:</label>
    <select name="occupation" id="occupation">
        <option value="student">Student</option>
        <option value="professional">Professional</option>
        <option value="other">Other</option>
    </select>
    
    <!-- Topics of Interest Checkboxes -->
    <legend>Topics Of Interest</legend>
    <label for="html">HTML</label>
    <input type="checkbox" name="checkbox" id="html">
    <label for="css">CSS</label>
    <input type="checkbox" name="checkbox" id="css">
    <label for="js">JS</label>
    <input type="checkbox" name="checkbox" id="js">
    
    <!-- Comments Section -->
    <label for="textarea1">Comments:</label>
    <textarea name="textarea" id="textarea1" cols="30" rows="5">Write Your Reviews...</textarea>
    
    <!-- Register Button -->
    <input type="submit" name="register" value="Register!">
</form>
