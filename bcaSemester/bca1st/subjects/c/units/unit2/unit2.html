Storage Classes

Overview of storage classes in C:

  • Automatic: Default storage class for local variables.
  • Register: Suggests storing variables in CPU registers for faster access.
  • Static: Retains the variable's value across function calls.
  • External: Used for global variables accessible across multiple files.
