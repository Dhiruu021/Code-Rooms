Overview of Flexbox: Column-Reverse Layout

In this example, the container is using the Flexbox layout with the flex-direction: column-reverse property. This causes the items inside the container to be displayed in a column, but in reverse order, with the last item appearing first.

Flexbox Layout Explanation

The .container is set to a Flex container with the flex-direction: column-reverse; property, which reverses the order of the child elements:


.container {
    display: flex;
    flex-direction: column-reverse;  /* Flex items in column order but reversed */
}

The .box elements are the children of the Flex container. Each .box has a set size and margin to create spacing between the boxes:


.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

Effect of flex-direction: column-reverse

With flex-direction: column-reverse;, the elements inside the container are stacked vertically, but the last element (in normal flow) will appear at the top, and the first element will appear at the bottom.

Use Cases for Flexbox: Column-Reverse

  • Reversing the order of elements in a layout without changing the HTML structure.
  • Creating dynamic layouts that adjust based on content, where the order needs to be reversed for specific visual effects.
  • Flexible navigation or message layout where the most recent items appear at the top.

This example demonstrates how the flex-direction: column-reverse property can be used to reverse the visual stacking of elements in a Flex container.

CSS Module for Flexbox: Column-Reverse

            
/* Flex container with column-reverse direction */
.container {
    display: flex;
    flex-direction: column-reverse; /* Reverses the order of elements */
}

/* Box styling */
.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

/* Individual box colors */
#box101 {
    background-color: aquamarine;
}
#box102 {
    background-color: rgb(112, 12, 80);
}
#box103 {
    background-color: rgb(128, 168, 18);
}
#box104 {
    background-color: rgb(27, 4, 78);
}

Overview of Flex Wrap

Flexbox is used to layout elements in a flexible way. When you apply display: flex; to a container, its child elements will be placed in a row by default. However, when there is not enough space to fit all the elements in a single row, you can use flex-wrap to wrap the items to the next line, instead of squishing them together.

By default, Flexbox items will try to fit in a single line. The flex-wrap property enables the items to wrap onto multiple lines if needed. This is especially useful for responsive layouts.

Flex Wrap: Available Values

The flex-wrap property accepts the following values:

  • nowrap: Prevents the items from wrapping, and they will shrink to fit the container if needed.
  • wrap: Allows the items to wrap to the next line if there's insufficient space in the container.
  • wrap-reverse: Similar to wrap, but the items will wrap to the previous line, in reverse order.

Practical Example: Flex Wrap

In this example, the .container uses flex-wrap: wrap; to ensure that the boxes will wrap to the next line when there's not enough space on the page. The boxes have a fixed size and will wrap if the container width is decreased:


.container {
    display: flex;
    flex-wrap: wrap; /* Allows the boxes to wrap to the next line */
}

.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

Shorthand Property: flex-flow

The flex-flow property is a shorthand for flex-direction and flex-wrap combined. It allows you to set both properties in a single line:


.container {
    display: flex;
    flex-flow: row wrap; /* Flex direction set to row and allows wrapping */
}

Use Cases for Flex Wrap

  • Creating flexible, responsive grids that adapt to different screen sizes.
  • Ensuring that elements like cards or images automatically wrap when there's insufficient space.
  • Implementing layout components like navigation menus that adjust according to the viewport width.

This example demonstrates how the flex-wrap property enables flexible layouts that adjust to varying screen sizes and available space.

CSS Module for Flex Wrap

.container {

What is Justify Content?

The justify-content property is used in Flexbox to align the items along the **main axis**. The main axis is determined by the flex-direction property (either row or column). In a row-based layout, the main axis runs horizontally, and in a column-based layout, it runs vertically.

Available Values for justify-content

Here are the available values for justify-content:

  • center: Aligns all the items to the center of the container.
  • flex-start: Aligns all the items to the start of the container (left for row or top for column).
  • flex-end: Aligns all the items to the end of the container (right for row or bottom for column).
  • space-between: Distributes the items evenly along the main axis, with the first item at the start and the last item at the end. The remaining space is distributed between the items.
  • space-around: Distributes the items evenly, with equal space around each item.
  • space-evenly: Distributes the items evenly with equal space between each item, including the space before the first item and after the last item.

Practical Example: Justify Content in Flexbox

The following example demonstrates how justify-content works by using different values to align the boxes inside the container. Try changing the value of justify-content in the CSS to observe the different alignments.


.container {
    display: flex;
    height: 300px;
    justify-content: space-evenly; /* Adjust the spacing between the items */
}

.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

Effect of justify-content on Flex Items

In this example, the items inside the container are aligned using justify-content: space-evenly;, which distributes the boxes evenly across the main axis, ensuring equal spacing between each box, including the first and last boxes.

What is Align Items?

The align-items property in Flexbox is used to align flex items along the **cross-axis**, which is typically the vertical axis when the flex direction is row (default) or the horizontal axis when the flex direction is column. This property helps in controlling how the items are aligned within the container.

Available Values for align-items

Here are the available values for align-items:

  • flex-start: Aligns the items to the start of the cross-axis (top for row, left for column).
  • flex-end: Aligns the items to the end of the cross-axis (bottom for row, right for column).
  • center: Aligns the items to the center of the cross-axis (vertically centered for row or horizontally centered for column).
  • stretch: Stretches the items to fill the container along the cross-axis. This only works if no specific height (for rows) or width (for columns) is defined for the flex items.

Practical Example: Align Items in Flexbox

The following example demonstrates how align-items works by aligning the items at the end of the container along the cross-axis using align-items: flex-end;.


.container {
    display: flex;
    height: 300px;
    align-items: end; /* Align the items to the bottom of the container */
}

.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

Effect of align-items on Flex Items

In this example, the flex items are aligned at the **bottom** of the container because of the use of align-items: end;. You can modify this to flex-start, center, or stretch to see different effects.

What is Align Content?

The align-content property is used in Flexbox to manage the spacing between lines of flex items when the container has multiple lines (such as when flex-wrap is used). It aligns the entire lines of items along the **cross-axis** (perpendicular to the main axis). This property only has an effect when the items are wrapped (i.e., when flex-wrap is set to wrap).

Available Values for align-content

Here are the available values for align-content:

  • flex-start: Aligns the lines to the start of the cross-axis (top for row direction, left for column direction).
  • flex-end: Aligns the lines to the end of the cross-axis (bottom for row direction, right for column direction).
  • center: Aligns the lines to the center of the cross-axis.
  • space-between: Distributes the lines evenly, with the first line at the start and the last one at the end of the cross-axis, leaving equal spaces between them.
  • space-around: Distributes the lines evenly with equal space around each line, including at the start and end.
  • stretch: Stretches the lines to fill the available space along the cross-axis.

Practical Example: Align Content in Flexbox

The following example demonstrates how align-content works with a wrapped flex container to align the content (lines) in the container.


.container {
    display: flex;
    flex-wrap: wrap;
    height: 1200px;
    border: 5px solid black;
    align-content: start; /* Aligns all the lines to the start of the container */
}

.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

Effect of align-content on Flex Items

In this example, the align-content property is used to remove gaps between lines of items by aligning them to the start of the container. As the screen size changes and the flex items wrap, the lines will be aligned accordingly.

Properties for Flex Items

In Flexbox, items can be customized with different properties that allow for control over their layout and behavior. The key properties used for flex items include order, flex-shrink, flex-grow, and flex-basis.

1. order

The order property is used to change the visual order of flex items. By default, all items have an order of 0, but you can change this value to rearrange the sequence of items. For example, setting order: -1 on an item will make it appear first, even if it is listed later in the HTML.

2. flex-shrink

The flex-shrink property specifies how much an item should shrink relative to other items when the container's width is reduced (or when there isn't enough space). The default value is 1, meaning the items will shrink equally. If set to a higher value, the item will shrink faster than others. For example, flex-shrink: 4 means the item will shrink four times faster than the other items.

3. flex-grow

The flex-grow property determines how much a flex item will grow relative to other items when there is extra space in the container. For instance, if flex-grow: 5 is set, the item will grow to fill available space faster than items with a lower flex-grow value.

4. flex-basis

The flex-basis property defines the initial size of a flex item before any extra space is distributed. It can be set to a fixed value (e.g., flex-basis: 400px) or left as auto for the item to use its natural size.

Shorthand Notation

These properties can be combined into one shorthand property:


flex: flex-grow flex-shrink flex-basis;

Example of Flexbox Properties

The following example demonstrates how to use these properties to manipulate the behavior and layout of flex items:

            
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 300px;
}

.box {
    width: 150px;
    height: 150px;
    padding: 10px;
    margin: 10px;
}

#box101 {
    background-color: aquamarine;
    order: 500;  /* Moves box1 to the last position */
    flex-shrink: 4;  /* Shrinks box1 faster than others */
}

#box102 {
    background-color: rgb(112, 12, 80);
}

#box103 {
    background-color: rgb(128, 168, 18);
    order: -1;  /* Moves box3 to the first position */
}

#box104 {
    background-color: rgb(27, 4, 78);
    order: 200;  /* Doesn't change order significantly */
    flex-grow: 5;  /* Makes box4 grow faster to cover available space */
}

Summary of Properties:

  • order: Changes the position of an item in the flex container.
  • flex-shrink: Controls the shrink speed of the item when the container shrinks.
  • flex-grow: Determines how much an item grows when there is extra space in the container.
  • flex-basis: Defines the initial size of an item before growing or shrinking.

FLEXBOX: USING ALIGN SELF

What is align-self?

The align-self property allows you to align an individual flex item along the cross-axis (the vertical axis by default) within a flex container. This property overrides the align-items property that applies to all flex items within the container.

Values of align-self:

  • center: Centers the item along the cross-axis.
  • start: Aligns the item at the start of the cross-axis (top by default).
  • end: Aligns the item at the end of the cross-axis (bottom by default).

Note: This property is used when you need to align an individual item differently from the other items in the flex container.

Example of align-self in Action:

The following example demonstrates the use of align-self to adjust the alignment of specific flex items:

            
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
      .container {
        display: flex;
        height: 1200px;
        padding: 10px;
        margin: 10px;
      }
      .box {
        display: block;
      }
      #box101 {
        background-color: aquamarine;
        align-self: center;  /* Centers box1 vertically */
      }
      #box102 {
        background-color: rgb(112, 12, 80);
        align-self: self-end;  /* Aligns box2 at the bottom */
      }
      #box103 {
        background-color: rgb(128, 168, 18);
      }
      #box104 {
        background-color: rgb(27, 4, 78);
      }
    </style>
  </head>
  <body>
    <div class="container">
      <div class="box" id="box101">box1</div>
      <div class="box" id="box102">box2</div>
      <div class="box" id="box103">box3</div>
      <div class="box" id="box104">box4</div>
    </div>
  </body>
</html>

Summary of align-self Values:

  • center: Centers the flex item along the cross-axis.
  • start: Aligns the item at the start of the cross-axis (usually the top).
  • end: Aligns the item at the end of the cross-axis (usually the bottom).

Practical Use Case:

The align-self property is particularly useful when you want to align one flex item differently from the others in a flex container without affecting the alignment of other items. For example, centering one item while others align at the top or bottom.
