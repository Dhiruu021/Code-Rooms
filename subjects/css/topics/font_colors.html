Overview of Font Styling in CSS

Font styling in CSS allows control over text appearance, such as size, weight, style, and spacing, to improve readability and aesthetic.

Font Properties

font-size: Sets the size of the font. For example:


p {
    font-size: 30px;
}

This font-size style will make the text in all <p> tags appear larger (30 pixels).

font-weight: Sets the thickness of the font. The following code makes text bold:


p {
    font-weight: bold;
}

Now all <p> elements will have a bold weight, enhancing emphasis.

font-style: Controls the style, like italic or oblique for slanted text:


article {
    font-style: oblique;
}

Setting font-style: oblique; will give the <article> text a slanted appearance.

line-height: Defines the spacing between lines, improving readability:


p {
    line-height: 70px;
}

With line-height set to 70 pixels, there will be 70 pixels of space between each line in <p> elements.

font-family: Specifies the typeface, like cursive for a handwritten look:


article {
    font-family: cursive;
}

This sets a cursive style to <article> text.

font shorthand: The font property allows multiple settings in one line:


#para {
    font: small-caption;
}

This shorthand applies a small-caps style to the <p id="para"> element.

Using External Fonts

To use a custom font from Google Fonts, follow these steps:

  1. Copy the font link from Google Fonts and add it in the <head> tag of your HTML file.
  2. Apply the font to elements by setting the font-family property in CSS, like so:

body {
    font-family: 'Your Font Name', sans-serif;
}

Overview of CSS Color Styling

In CSS, colors can be applied in various ways, such as named colors, RGB, RGBA, and hexadecimal formats. Each method allows control over the color's intensity, transparency, and visibility.

Named Colors

CSS provides a set of predefined color names, like yellow or blue. These can be used directly:


p {
    color: yellow; /* Applies a yellow color to the paragraph text */
}

RGB Colors

The RGB format uses red, green, and blue values to create a wide range of colors. Each color can range from 0 to 255. For example:


article {
    color: rgb(55, 122, 156); /* Applies a custom RGB color to the article text */
}

RGBA Colors

The RGBA format adds an alpha value for transparency. The alpha value ranges from 0 (fully transparent) to 1 (fully opaque):


article {
    background-color: rgba(255, 100, 122, 0.2); /* Adds a semi-transparent background */
}

Hexadecimal Colors

Hex colors begin with a # symbol, followed by six characters representing red, green, and blue. You can use an online color picker to find hex codes:


#article2 {
    background-color: #ffffff; /* White background */
    color: #0fd05f; /* Custom green text color */
}
