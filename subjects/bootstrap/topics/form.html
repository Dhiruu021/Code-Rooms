Bootstrap Forms

Overview

Bootstrap provides a wide range of classes to style forms and form components easily. Using Bootstrapâ€™s form controls, you can quickly style inputs, labels, checkboxes, switches, and select elements, enhancing both functionality and visual consistency.

Basic Form Structure

A basic form layout can be created using Bootstrap classes. Adding form-control to input elements ensures they take up the full width of their container, while mb-3 or mb-4 classes help manage spacing.

Examples of Form Components

Basic Text Input


<form>
    <label for="email" class="form-label">Email</label>
    <input type="email" class="form-control" id="email" placeholder="Enter your email">
</form>

This form example includes a labeled email input field styled with form-control for a full-width appearance and consistent padding.

Password Input with Label


<form>
    <label for="password" class="form-label">Password</label>
    <input type="password" class="form-control" id="password" placeholder="Enter your password">
</form>

Password inputs can also use the form-control class, creating a uniform look across different input types.

Different Input Sizes


<form>
    <input type="text" class="form-control-lg" placeholder="Large Input">
    <input type="text" class="form-control-sm" placeholder="Small Input">
</form>

Use form-control-lg for larger inputs and form-control-sm for smaller inputs, allowing customization based on the form's design requirements.

Dropdown Selection


<select class="form-select">
    <option selected>Choose your city</option>
    <option value="1">City 1</option>
    <option value="2">City 2</option>
</select>

Dropdowns can be styled with form-select, providing a consistent look and feel for selection elements.

Checkboxes and Switches


<div class="form-check">
    <input class="form-check-input" type="checkbox" value="" id="flexCheckDefault">
    <label class="form-check-label" for="flexCheckDefault">Default checkbox</label>
</div>
<div class="form-check form-switch">
    <input class="form-check-input" type="checkbox" role="switch" id="flexSwitchCheckDefault">
    <label class="form-check-label" for="flexSwitchCheckDefault">Switch</label>
</div>

Use form-check and form-check-input for standard checkboxes, and form-check form-switch for switch-style checkboxes.

Conclusion

By leveraging Bootstrap's form controls, you can easily create accessible, responsive, and visually appealing forms. These controls simplify styling and layout adjustments, making form creation quicker and more consistent.

Bootstrap Grid Form Layouts

Overview

Bootstrap's grid system enables responsive form layouts by dividing the page into a flexible grid of rows and columns. This approach is especially useful for organizing form fields side-by-side on larger screens and stacking them on smaller devices.

Example: Basic Form with Rows and Columns

This example illustrates a two-column layout where the "Email" and "Password" fields are aligned side-by-side on medium and larger screens:


<form>   
    <div class="row">
        <div class="mb-4 col-md-6">
            <label for="email" class="form-label">Email</label> 
            <input type="email" class="form-control" id="email" placeholder="Enter your email">
        </div>

        <div class="mb-4 col-md-6">
            <label for="password" class="form-label">Password</label> 
            <input type="password" class="form-control" id="password" placeholder="Enter your password">
        </div>
    </div>
</form>

Grid System Classes Explained

  • row: Creates a horizontal group of columns, defining the layout structure.
  • col-md-6: Specifies the width of each column on medium and larger screens, creating a two-column layout.
  • mb-4: Adds bottom margin for spacing between form fields.

More Complex Layout with Switches and Checkboxes

This example demonstrates adding checkboxes and switches within the grid layout:


<form>   
    <div class="row">
        <div class="mb-4 col-4">
            <div class="form-check form-switch">
                <input type="checkbox" class="form-check-input" id="flexSwitchCheckDefault">
                <label class="form-check-label" for="flexSwitchCheckDefault">Switch</label>
            </div>
        </div>

        <div class="mb-4 col-2">
            <div class="form-check form-check-inline">
                <input type="checkbox" class="form-check-input" id="inlineCheckbox2" value="option2">
                <label class="form-check-label" for="inlineCheckbox2">2</label>
            </div>
        </div>
    </div>
</form>

Responsive Adjustments

The grid system in Bootstrap is responsive by default, adjusting the form layout based on screen size. Use classes like col-*, col-sm-*, col-md-*, etc., to control the number of columns for various screen sizes.

Conclusion

With the Bootstrap grid system, you can build responsive, multi-column layouts that adapt to various screen sizes, making forms more user-friendly and organized.
