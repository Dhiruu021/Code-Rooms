Basics & operators

Template Literals

Template literals allow for easy string interpolation and multi-line strings. They are enclosed by backticks (``). 


// Example of Template Literals
let pencilPrice = 10;
let erasorPrice = 5;
let output = `The total price is: ${pencilPrice + erasorPrice} Rupees`;
console.log(output); // Output: The total price is: 15 Rupees

Comparison Operations

JavaScript provides various comparison operators to compare values. 


// Comparison Operations
let a = 10;
let b = 20;
console.log(a > b);  // false
console.log(a < b);  // true
console.log(a >= b); // false
console.log(a <= b); // true

console.log(a == b);  // false
console.log(a != b);  // true

// Understanding Unicode in Comparison
let d = 5;
let e = "4";
console.log(d > e);   // true
console.log(d < e);   // false
console.log(d <= e);  // false
console.log(d >= e);  // true
console.log(d == e);   // true (type coercion)
console.log(d != e);   // false (type coercion)
console.log(d === e);  // false (strict equality)
console.log(d !== e);  // true (strict inequality)

Switch Case Statement

The switch statement evaluates an expression, matching the expression's value to a case clause. 


// Example of Switch Case
let color = "red";

switch (color) {
    case "red":
        console.log("Stop");
        break;
    case "yellow":
        console.log("Ready");
        break;  
    case "green":
        console.log("Go");
        break;
    default:
        console.log("Invalid light");
        break;

String Methods

trim()

The trim() method removes whitespace from both ends of a string. 


// Example of trim()
let namee = "       ujjawal       ";
console.log(namee);         // Output: "       ujjawal       "
console.log(namee.trim()); // Output: "ujjawal"

let password = prompt("Enter your password:");
let realPass = password.trim();
console.log(realPass); // Output: trimmed password

indexOf()

The indexOf() method returns the index of the first occurrence of a specified value in a string. Returns -1 if the value is not found. 


// Example of indexOf()
let supra = "hello ji kaise ho";
console.log(supra.indexOf("j"));        // Output: 6
console.log(supra.indexOf("kaise"));     // Output: 9
console.log(supra.indexOf("e"));         // Output: 1
console.log(supra.indexOf("z"));         // Output: -1

Chaining Multiple Methods

You can chain multiple string methods together, applying them left to right. 


// Chaining Example
let supraa = "    hello    ";
let newSupra = supraa.trim().toUpperCase();
console.log(newSupra); // Output: "HELLO"

slice(inclusive_idx, exclusive_idx)

The slice() method extracts a section of a string and returns it as a new string. It takes two parameters: the starting index (inclusive) and the ending index (exclusive). 


// Example of slice()
let supraa = "college";
let newSupra = supraa.slice(4, supraa.length); // "ge"
let supra2 = supraa.slice(-2);                 // "ge"
let supra3 = supraa.slice(1, -1);             // "oll"
console.log(newSupra);  // Output: "ge"
console.log(supra2);    // Output: "ge"
console.log(supra3);    // Output: "oll"

replace("old_value", "new_value") and
replaceAll("old_value", "new_value")

The replace() method replaces the first occurrence of a specified value in a string, while replaceAll() replaces all occurrences. 


// Example of replace() and replaceAll()
let supraa = "i love coding";
let newSupra = supraa.replace("love", "do");       // "i do coding"
let supra2 = supraa.replace("o", "x");              // "i lxve coding" (first occurrence)
let supra4 = supraa.replaceAll("o", "x");           // "i lxve cxding" (all occurrences)
let supra3 = supraa.slice("jojo", "jejo");         // Not Available
console.log(newSupra);  // Output: "i do coding"
console.log(supra2);    // Output: "i lxve coding"
console.log(supra4);    // Output: "i lxve cxding"
console.log(supra3);    // Output: undefined
console.log(typeof(supra3)); // Output: "undefined"

repeat(No._of_times)

The repeat() method constructs and returns a new string which contains the specified number of copies of the string on which it was called, concatenated together. 


// Example of repeat()
let str = "mango ";
console.log(str.repeat(3)); // Output: "mango mango mango "

Loops

For Loop

The for loop is a control flow statement for specifying iteration, allowing code to be executed repeatedly based on a boolean condition. It consists of three parts: initialization, condition, and update.


// Basic syntax of a for loop
// for(initialization; condition; update) {
//     // task
// }
for (let i = 1; i < 5; i++) {
    console.log(i); // Output: 1, 2, 3, 4
}

console.log('Even numbers between 1 and 15:');
for (let i = 2; i < 15; i = i + 2) {
    console.log(i); // Output: 2, 4, 6, 8, 10, 12, 14
}

let colors3 = ["blue", "red", "green", "orange", "yellow", "pink", "white"];
for (let i = 0; i < colors3.length; i++) {
    console.log(i, colors3[i]); // Outputs the index and color
}

For-of Loop

The for-of loop is a simpler way to iterate over iterable objects like arrays and strings. It directly gives you the value of each element, eliminating the need to use an index.


// For-of loop for strings and arrays
let colors = ["blue", "red", "green", "orange", "yellow", "pink", "white"];
for (let color of colors) {
    console.log(color); // Outputs each color
}

let heroes = [["king", "khan", "supra"], ["ironman", "spiderman", "thor"]];
for (let hero of heroes) {
    for (let single of hero) { // Nested for-of loop
        console.log(single); // Outputs each hero
    }
}

For-in Loop

The for-in loop iterates over the enumerable properties of an object. It is primarily used for objects, but it can also be used with arrays to access the indices.


// For-in loop for objects
let person = {name: "Ujjawal", age: 16, city: "Delhi"};
for (let key in person) {
    console.log(key, person[key]); // Outputs key-value pairs
}

// For-in loop for arrays
let fruits = ["banana", "apple", "papaya"];
for (let index in fruits) {
    console.log(index, fruits[index]); // Outputs index and fruit
}

While Loop

The while loop executes a block of code as long as the specified condition evaluates to true. It is useful when the number of iterations is not known beforehand.


let i = 1;
while (i < 5) {
    console.log(i); // Output: 1, 2, 3, 4
    i++;
}

console.log('Odd numbers between 1 and 15:');
let j = 1;
while (j < 15) {
    console.log(j); // Output: 1, 3, 5, 7, 9, 11, 13
    j += 2;
}

Do-While Loop

The do-while loop is similar to the while loop, but it guarantees that the code block will execute at least once, even if the condition is false initially.


let k = 1;
do {
    console.log(k); // Output: 1, 2, 3, 4
    k++;
} while (k < 5);

let l = 10;
do {
    console.log(l); // Output: 10
    l++;
} while (l < 10); // The loop will execute only once

This Keyword

The this keyword refers to the object that is executing the current piece of code. It is commonly used to access the properties (attributes) of an object within its methods.

Try & Catch

What is Try & Catch?

The try statement allows you to define a block of code to be tested for errors while it is being executed.

The catch statement allows you to define a block of code to be executed if an error occurs in the try block.

Why Use Try & Catch?

In a JavaScript file, if an error occurs in the code, it can stop the execution of the program, potentially causing the site to crash. To handle errors and exceptions gracefully without interrupting the entire execution, we use try and catch.

This approach is particularly useful in API calls.

Syntax


try {
    // code to test
} catch {
    // statement executed when an error is found
}

Example 1: Basic Try & Catch


console.log("hello");
console.log("hello");
try {
    console.log(a); // a is not defined
} catch {
    console.log("Found an error! But your below code is executing fine.");
}  
console.log("hello");
console.log("hello");
try {
    console.log("I am okay so catch is not executing (we are like if else)");
} catch {
    console.log("Found an error! But your below code is executing fine.");
}  
console.log("hello");

Example 2: Catching Errors with Error Objects


try {
    console.log(b); // b is not defined
} catch (err) {
    console.log(err); // Output the actual error message
}

Arrow Functions

What are Arrow Functions?

Arrow functions are a compact way to write functions without the need for the function keyword. They are also nameless.

It is always advisable to assign these functions to a variable or use them as callbacks (a function passed as an argument) in higher-order functions.

Syntax


const func = (arg1, arg2...) => {
    // function definition (actual task)
};

Example: Basic Arrow Function


// Arrow function
const sum = (a, b) => {
    console.log(`sum is = ${a + b}`);
};
sum(2, 4);
console.log(sum, "have you noticed I'm not with 'f' symbol, means I'm treating as a value of variable.");

// Normal function
const helloji = function() {
    console.log("helloji");
};
helloji();
console.log(helloji, "I have 'f' symbol because I'm a normal function.");

// Arrow function
const hello = () => {
    console.log("hello");
};
hello();
console.log(hello);

Notes

In an arrow function, if we have a single argument, then parentheses are optional. However, if we have two or more arguments or zero arguments, parentheses are necessary.

Implicit Return

If an arrow function only returns a value without doing any task, the return keyword can be omitted. This is called implicit return.


const mul = (a, b) => (a * b);
console.log(mul(2, 3)); // Output: 6
console.log(mul(2, 19)); // Output: 38

This Keyword in Arrow Functions

There is a significant difference in how this is scoped in arrow functions compared to normal functions:

Arrow functions inherit this from their parent scope, whereas normal functions have their own this based on the object that called them.


const student = {
    name: "Ujjawal",
    marks: 95,
    prop: this, // global scope (this = window object)

    getName: function() { // normal function
        console.log(this); // here, this = student (scope of parent calling object)
        return this.name;
    },

    getMarks: () => { // arrow function
        console.log(this); // here, this = window (global scope)
        return this.marks;
    }
};

console.log(student);
console.log(student.prop); // Output: window object

console.log(student.getName()); // calling the normal function
console.log(student.getMarks()); // calling the arrow function (undefined)

Useful Case of This Keyword in Arrow Functions


const student2 = {
    name: "Akhand",
    marks: 96,
    prop: this,

    getInfo1: function() {
        setTimeout(() => {
            console.log(this); // this = student (lexical scope of nested function)
        }, 2000);
    },

    getInfo2: function() {
        setTimeout(function() {
            console.log(this); // this = window (global scope)
        }, 2000);
    }
};

console.log(student2.getInfo1()); // calling getInfo1
console.log(student2.getInfo2()); // calling getInfo2

Summary

In summary:

  • Arrow Function: Inherits this from its parent scope.
  • Normal Function: this is defined by the object that calls the function.

Spread

Spread

The spread operator expands an iterable (like arrays and strings) into multiple values.

Syntax


function func(...arr) {
    // tasks
}

Why Spread?

Example: Finding the minimum element of an array.

Normally


let arr = [1, 2, 5, 4, 6, 7, 89, 6, 45];
let minimum = Math.min(arr[0], arr[1], arr[2], arr[3], arr[4], arr[5], arr[6], arr[7], arr[8]);
console.log(minimum);

Using Spread


minimum = Math.min(...arr);
console.log(minimum);

maximum = Math.max(...arr);
console.log(maximum);

Print All Iterated Values


console.log(...arr); // prints all iterated values of an array
console.log(..."ujjawal jaiswal"); // separated and printed

The spread operator is basically used to spread the elements.

Using Spread with Array Literals

let all = [...odd, ...even]; // combining odd and even arrays

Rest

Rest

The rest operator allows a function to take an indefinite number of arguments and bundle them into an array.

Opposite to Spread

  • Spread: A single unit to multiple values (spreading elements).
  • Rest: Multiple values combined into a single unit.

Syntax + Example


function sum(...args) {
    return args.reduce((add, el) => add + el);
}

Example of Print Function


function print(...arg) {
    for (let i = 0; i < arg.length; i++) {
        console.log("argument is:", arg[i]);
    }
}
console.log(print); // function reference

print(3);
print(12, 2, 3, 4);
print("hello", "ji", "kaise ho");
print(1, 3, 4, 6, 4, 5, "ujjawal", "arr", 78.0); // can pass any number of arguments

Arguments Object

There is an inbuilt "arguments collector" in JavaScript that collects all the passed arguments for that function.


function min(a, b, c, d) {
    console.log(arguments);
    console.log("length of argument attribute:", arguments.length);
}
min();
min(1, 2, 3, 4); // Note: arguments is not an array, hence cannot use array methods
min(1, 2, 3, 4, 5, 6, 7, 8, 9); // length of arguments is not dependent on the length of parameters

Use of Rest


function addAll(...args) {
    return args.reduce((add, el) => add + el);
}
console.log("Addition of All elements is:", addAll(1, 2));
console.log("Addition of All elements is:", addAll(1, 2, 3, 4, 5, 6, 7));
console.log("Addition of All elements is:", addAll(1, 2, 3, 4, 5, 6, 7, 55, 44, 3, 4, 56, 7));

function max(...restArray) {          
    return restArray.reduce((max, el) => {
        if (max < el) {
            return el;
        } else {
            return max;
        }
    });
}
console.log("Maximum of All elements is:", max(1, 2, 3, 4, 5));
console.log("Maximum of All elements is:", max(1, 2, 3, 4, 5, 6, 6, 77, 3, 4, 7, 8, -1243, 34));

Note

If you want to provide some arguments in a function but do not want to count them into the rest array, write them before the arguments array.


function addAll(a, b, c, ...args) { // now a, b, c are not part of args array
    return args.reduce((add, el) => add + el);
}

Destructuring

Destructuring

Destructuring allows you to unpack values from arrays or properties from objects into distinct variables.

Destructuring Arrays

Storing values of an array into multiple variables.


let names = ["ujjawal", "shivam", "prince", "kunnu", "nikhil", "shubham", "vikas"]; 

// Method 1
let winner = names[0];
let runnerUp = names[1];
let secondRunnerUp = names[2];

console.log(winner);
console.log(runnerUp);
console.log(secondRunnerUp);

// Method 2 (Destructuring)
let [winner1, runnerUp1, secondRunnerUp1] = names; // Assign from the start of the array.
console.log(winner1);
console.log(runnerUp1);
console.log(secondRunnerUp1);        

// Using destructuring with rest for storing all other names in a new array.
let [winner2, runnerUp2, ...others] = names;
console.log(winner2);
console.log(runnerUp2);
console.log(others);

Destructuring with Objects

Storing all the values of an object in variables. The key becomes the variable name.


const student = {
    name: "karan",
    age: 16,
    class: 12,
    subjects: ["hindi", "english", "math", "science"],
    username: "karan@2323",
    password: "********"
};

// Destructuring
let { username, password } = student; // It will search for username and password in the object as keys.
console.log(username);
console.log(password);

// If we want to store in a different variable name other than the key name.
let { name, username: new_variable, password: pass, ifNotExistInObject = "pune" } = student; // Storing values in new variables.
console.log(name);
console.log(new_variable);
console.log(pass);
console.log(ifNotExistInObject); // If not present in the object, we can also add a default value.
