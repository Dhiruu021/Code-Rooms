Bootstrap Badges and Notifications

Overview

Badges are used to display a small count or status indicator. They can be used with various Bootstrap components, such as buttons and headings, to enhance the visibility of certain information. This section covers different uses of badges in Bootstrap.

Basic Badges


<h3>Notifications <span class="badge text-bg-secondary">9+</span></h3>
<h3>Example heading <span class="badge text-bg-danger">New</span></h3>
        
<span class="badge text-bg-danger">101+</span>

The above examples showcase basic usage of badges with headings. Badges can be styled using Bootstrap's contextual classes such as text-bg-secondary and text-bg-danger.

Badges in Buttons


<button class="btn btn-primary">Followers
    <span class="badge bg-dark">100+ </span>
</button>

<button class="btn btn-primary">Followers
    <span class="badge bg-dark rounded-pill">100+ </span>
</button>

Badges can also be added inside buttons to show counts or notifications. The rounded-pill class provides a pill-shaped badge for a different visual appearance.

Positioning Badges

Bootstrap allows for absolute positioning of badges, enabling custom placements.


<button type="button" class="btn btn-primary position-relative">
    New Followers
    <span class="position-absolute top-0 start-100 translate-middle badge rounded-pill bg-danger">
      99+
      <!-- <span class="visually-hidden">unread messages</span> -->
    </span>
</button>

In the example above, the badge is positioned relative to the button. The position-absolute class is used to position the badge at the top-right corner of the button. The translate-middle class adjusts its position to center it over the corner.

Badges with Icons


<button type="button" class="btn btn-primary position-relative">
    Ujjawal Jaiswal
    <span class="position-absolute top-0 start-100 translate-middle p-2 bg-danger border border-light rounded-circle">
      <span class="visually-hidden">New alerts</span>
    </span>
</button>

Badges can also include icons or other indicators. In this case, an icon could be represented using a circular badge alongside a text button, enhancing user engagement.

Conclusion

Badges in Bootstrap provide a simple yet effective way to convey additional information alongside components. By utilizing various classes, you can create visually appealing notifications that enhance user experience.
