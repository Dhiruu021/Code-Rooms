Bootstrap Basic Grid

Overview

Bootstrap's grid system allows for responsive design using a series of containers, rows, and columns to layout and align content. It is built with flexbox and is fully responsive, making it easy to create complex layouts across different devices.

Basic Structure of the Grid

The grid system is based on a 12-column layout. You can specify how many columns each element should span, allowing for various layouts. 


<div class="container">
    <div class="row">
        <div class="col bg-danger">1 unit</div>
        <div class="col bg-success">1 unit</div>
        <div class="col bg-primary">1 unit</div>
        <div class="col bg-secondary">1 unit</div>
        <div class="col bg-danger">1 unit</div>
        <div class="col bg-success">1 unit</div>
        <div class="col bg-primary">1 unit</div>
        <div class="col bg-secondary">1 unit</div>
        <div class="col bg-danger">1 unit</div>
        <div class="col bg-success">1 unit</div>
        <div class="col bg-primary">1 unit</div>
        <div class="col bg-secondary">1 unit</div>
    </div>
</div>

In this example, we create a row with 12 equal columns. Each column can be styled with different Bootstrap background utility classes.

Responsive Columns

Bootstrap allows you to create responsive columns that adjust based on the screen size. You can specify different column sizes for various breakpoints. For example: 


<div class="row">
    <div class="col-6 bg-secondary">one 50% 6-units</div>
    <div class="col-6 bg-danger">two 50% 6-units</div>
</div>

Here, we create two columns that each take up 50% of the width on larger screens. On smaller screens, they will stack vertically.

Column Sizing

Columns can be sized using the grid classes, allowing you to specify the width in units. For example: 


<div class="row">
    <div class="col-2 bg-danger">2 units</div>
    <div class="col-6 bg-success">6-units</div>
    <div class="col-4 bg-primary">4-units</div>
</div>

In this example, the first column takes 2 units, the second column takes 6 units, and the last one takes 4 units.

Equal Columns

To create equal-width columns, simply use the .col class without any specified size: 


<div class="row">
    <div class="col bg-danger">Equal size</div>
    <div class="col bg-success">of columns</div>
    <div class="col bg-primary">in a row</div>
</div>

Each column will take up an equal amount of space in the row, adjusting to fit the screen size.

Conclusion

Bootstrap's grid system is a powerful tool for creating responsive layouts. By utilizing containers, rows, and columns, you can easily design web pages that adapt to different screen sizes while maintaining a consistent structure.

Grid Options

Overview

Bootstrap's grid system provides multiple options to customize the layout based on different screen sizes. You can use responsive classes to adjust column sizes at various breakpoints, ensuring your layout adapts to different devices.

Basic Grid Options

You can define grid columns using different responsive classes. For example, the classes .col-sm-*, .col-md-*, .col-lg-*, and .col-xl-* allow you to specify how many columns an element should occupy on different screen sizes.

Examples of Grid Options

Two Equal Columns on Extra Large Screens


<div class="row">
    <div class="col-xl-6 bg-secondary">one 50%</div>
    <div class="col-xl-6 bg-danger">two 50%</div>
</div>

In this example, on extra-large screens, we have two equal columns, each taking up 50% of the width.

Responsive Columns for Medium Screens


<div class="row">
    <div class="col-md-2 bg-danger">when hits medium size</div>
    <div class="col-md-8 bg-success">of display it will take</div>
    <div class="col-md-2 bg-primary">full width.</div>
</div>

Here, when the screen size hits medium size, the first column takes 2 units, the second takes 8 units, and the last takes 2 units, effectively occupying the full width of the row.

Combining Grid Options for Better Responsiveness


<div class="row">
    <div class="col-md-6 col-xl-4 bg-danger">Lorem, ipsum dolor sit amet consectetur adipisicing elit.</div>
    <div class="col-md-6 col-xl-4 bg-success">Lorem ipsum dolor, sit amet consectetur adipisicing elit.</div>
    <div class="col-md-6 col-xl-4 bg-primary">Lorem ipsum dolor sit amet consectetur adipisicing elit.</div>
</div>

This example demonstrates how to use multiple grid options in a single row. The columns will take up 6 units on medium screens and 4 units on extra-large screens, allowing for a responsive design that adjusts seamlessly across different devices.

Conclusion

Utilizing Bootstrap's grid options allows you to create flexible and responsive layouts. By combining different column sizes at various breakpoints, you can ensure your web page looks great on any device.

Grid Vertical Alignment

Overview

Bootstrap's grid system allows for vertical alignment of columns within a row. This feature is particularly useful for centering content or distributing space within the columns vertically, enhancing the overall layout and user experience.

Basic Vertical Alignment Options

You can align columns vertically using utility classes such as .align-start, .align-center, and .align-end. These classes can be applied to the row element to control the vertical alignment of its columns.

Examples of Grid Vertical Aligning

Aligning Columns to the Top


<div class="row align-items-start">
    <div class="col bg-secondary">Column 1: Top Aligned</div>
    <div class="col bg-danger">Column 2: Top Aligned</div>
</div>

In this example, both columns are aligned to the top of the row, ensuring they start from the same vertical position.

Centering Columns Vertically


<div class="row align-items-center">
    <div class="col bg-success">Column 1: Center Aligned</div>
    <div class="col bg-primary">Column 2: Center Aligned</div>
</div>

Here, the columns are centered vertically within the row, providing a balanced look regardless of their height.

Aligning Columns to the Bottom


<div class="row align-items-end">
    <div class="col bg-warning">Column 1: Bottom Aligned</div>
    <div class="col bg-info">Column 2: Bottom Aligned</div>
</div>

This example demonstrates how to align columns to the bottom of the row, making it easier to create layouts with different content heights while maintaining alignment.

Conclusion

Utilizing vertical alignment options in Bootstrap's grid system enhances the layout flexibility and visual appeal of your design. By strategically aligning content, you can achieve a more organized and aesthetically pleasing webpage.
