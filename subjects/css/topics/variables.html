Overview of CSS Variables

CSS Variables, also known as Custom Properties, allow you to store reusable values in CSS. These variables can be used throughout the CSS file and can be updated globally, making your styles more maintainable and easier to manage.

How to Declare a CSS Variable

To declare a CSS variable, use the following syntax:


--variable-name: value;

For example:


--primary-color: brown;
--custom-width: 330px;

In the above example, two variables are declared: --primary-color and --custom-width, with values of brown and 330px, respectively.

How to Use a CSS Variable

To use a variable, apply the var() function with the variable name:


property: var(--variable-name);

For example:


width: var(--custom-width);
border: 1px solid var(--primary-color);

The var(--custom-width) will fetch the value of 330px, and var(--primary-color) will use the color brown for the border.

Scope of CSS Variables

By default, CSS variables are scoped to the element in which they are declared. This means if you declare a variable inside a class (like .box), it will only be accessible within that class and its descendants.

Global CSS Variables

If you want to declare a variable that can be accessed anywhere in the stylesheet, declare it inside the :root pseudo-class:


:root {
    --primary-color: brown;
    --custom-width: 330px;
}

The :root selector refers to the highest-level parent element (the <html> element), so variables declared here are accessible anywhere in the document.

Example of Global Variables

With the :root declaration, you can use the variables throughout your styles:


.container {
    background-color: beige;
    height: 650px;
    border: 2px solid black;
    display: flex;
    justify-content: center;
    align-items: center;
    width: var(--custom-width);
}

Advantages of Using CSS Variables

  • Maintainability: You can change the value of a variable in one place, and it will automatically update everywhere it is used.
  • Consistency: Variables help ensure that repeated values (like colors, sizes, or fonts) are consistent across your website.
  • Dynamic Updates: You can dynamically change variables with JavaScript, allowing for more interactive designs.

Notes on CSS Variables

  • CSS variables are case-sensitive.
  • CSS variables can inherit from parent elements if declared inside them.
  • They are also inherited by child elements, which is useful for theming.

Summary of CSS Variables

CSS variables make it easier to manage your styles by allowing you to define reusable values and update them globally across your stylesheets. Whether you're working on a small project or a large-scale website, CSS variables can improve both maintainability and flexibility.
