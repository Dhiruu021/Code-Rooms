What is CSS Animation?

CSS Animation allows you to change an element's style from one state to another over a specified duration. With the help of keyframes, you can define the start and end points of an animation, and use additional properties to control the timing, delay, and iteration behavior of the animation.

Properties in CSS Animation

The following are the key properties used for CSS animations:

  • @keyframes
  • animation-name
  • animation-duration
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-timing-function
  • animation-fill-mode
  • animation-play-state

Syntax for Keyframes

Keyframes define the stages of the animation. You specify the styles at different points using from (start) and to (end) or percentage-based positions.


@keyframes animation-name {
    from {
        /* starting style */
    }
    to {
        /* ending style */
    }
}

Example of Keyframes

In this example, we define an animation that moves a box from left to right:


.box {
    animation-name: rightMovement;
    animation-duration: 5s;
}

@keyframes rightMovement {
    from {
        left: 0;
    }
    to {
        left: 200px;
    }
}

CSS Animation Properties

Here are the essential properties used to control the behavior of animations:

  • animation-name: Specifies the name of the animation (defined in the @keyframes rule). animation-name: rightMovement;
  • animation-duration: Specifies the duration of the animation. animation-duration: 5s;
  • animation-iteration-count: Specifies how many times the animation will run. animation-iteration-count: 2;
  • animation-delay: Specifies the delay before the animation starts. animation-delay: 2s;
  • animation-timing-function: Defines the speed curve of the animation. Example values:
    • linear: Constant speed.
    • ease: Starts slow, speeds up, and slows down again (default).
    • ease-in: Starts slow, then speeds up.
    • ease-out: Starts fast, then slows down.
    • ease-in-out: Starts and ends slow.
    • cubic-bezier(n,n,n,n): Define your own speed curve.
  • animation-direction: Specifies the direction of the animation. Values:
    • normal: Runs from start to end.
    • reverse: Runs from end to start.
    • alternate: Runs forward, then reverses.
    • alternate-reverse: Runs in reverse, then forward.
  • animation-fill-mode: Specifies the state of the element before or after the animation. Values:
    • forwards: Keeps the element in the final state after the animation ends.
    • backwards: Keeps the element in the initial state before the animation starts.
    • both: Applies both the forward and backward behaviors.
    • none: No behavior applied.
  • animation-play-state: Specifies whether the animation is running or paused. Values:
    • paused: The animation is paused.
    • running: The animation is active.

Shorthand Notation for CSS Animation

You can combine all the animation properties into one shorthand property:


animation: duration timing-function delay iteration-count direction fill-mode play-state name;

Example of shorthand animation syntax:


animation: 5s linear 1s infinite alternate none paused rightMovement;

Conclusion

CSS Animations provide a powerful way to add motion to your website. By using keyframes and various animation properties, you can create smooth and dynamic effects. Understanding the key properties and their shorthand usage will help you efficiently implement animations on your site.

CSS Animation Example

Sample HTML Structure:

            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<style>
    * {
        margin: 0;
        padding: 0;
        box-sizing: border-box;
    }
    .container {
        width: 100vw;
        height: 100vh;
        background-color: beige;
        border: 5px solid black;
    }
    .box {
        width: 350px;
        height: 80px;
        background-color: blue;
        border: 3px solid yellow;
        margin: 10px;
        position: relative;
        animation-name: rightMovement;
        animation-duration: 5s;
        animation-iteration-count: infinite;
        animation-delay: 2s;
        animation-timing-function: linear;
    }
    @keyframes rightMovement {
        from {
            top: 0;
            left: 0;
        }
        to {
            top: 0;
            left: 1200px;
        }
    }
</style>
<body>
    <div class="container">
        <div class="box">
            hello g i'm going away 1200px from left side.
        </div>
    </div>
</body>
</html>

What is this example?

This example demonstrates how to apply CSS animations to an element. The blue box moves from the left side of the screen to the right side with a specified duration and timing.

Explanation of Key Parts:

1. Global Reset

The following CSS resets the margin, padding, and box-sizing properties globally for all elements, ensuring consistent spacing and layout:


* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2. Container

The container is set to fill the entire viewport width and height, with a beige background and a border:


.container {
    width: 100vw;
    height: 100vh;
    background-color: beige;
    border: 5px solid black;
}

3. Box Styling

The box element has a blue background, yellow border, and is given a margin. It is positioned relative to allow for animation changes to its position:


.box {
    width: 350px;
    height: 80px;
    background-color: blue;
    border: 3px solid yellow;
    margin: 10px;
    position: relative;
}

4. Animation Properties

The animation is applied to the box element with these key properties:

  • animation-name: Specifies the name of the animation, in this case, rightMovement.
  • animation-duration: Defines how long the animation will run, here it is 5s.
  • animation-iteration-count: Defines how many times the animation will repeat. In this case, it is set to infinite for an endless loop.
  • animation-delay: Adds a delay before the animation starts, set to 2s.
  • animation-timing-function: Controls the speed of the animation. Here it is set to linear to maintain constant speed throughout the animation.

5. @keyframes Rule

The @keyframes rule defines the animation behavior at different points in time. In this case, the box moves horizontally from left to right:


@keyframes rightMovement {
    from {
        top: 0;
        left: 0;
    }
    to {
        top: 0;
        left: 1200px;
    }
}

What happens in this example?

When the page is loaded, the box element will start at the left (top: 0, left: 0) and move horizontally to the right (top: 0, left: 1200px) over 5 seconds. This will repeat infinitely with a 2-second delay before each iteration starts.

Conclusion

This example shows how to apply a simple CSS animation that moves an element across the screen. You can customize the animation duration, timing, iteration count, and direction by adjusting the relevant CSS properties. Understanding how to use @keyframes and the different animation properties gives you the ability to create dynamic and interactive effects in your web designs.

What is this example?

This example demonstrates the use of the animation-direction: alternate; property to make an animation reverse direction after each cycle, creating a back-and-forth movement for the animated element.

Explanation of Key Parts:

1. Global Reset

The following CSS resets the margin, padding, and box-sizing properties globally for all elements, ensuring consistent spacing and layout:


*{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2. Container

The container fills the entire viewport width and height, with a beige background and a black border:


.container{
    width: 100vw;
    height: 100vh;
    background-color: beige;
    border: 5px solid black;
}

3. Box Styling

The box element has a blue background, yellow border, margin, and is positioned relatively to allow the animation to affect its position:


.box{
    width: 350px;
    height: 80px;
    background-color: blue;
    border: 3px solid yellow;
    margin: 10px;
    position: relative;
}

4. Animation Properties

The animation is applied to the box element with these key properties:

  • animation-name: Specifies the name of the animation, here it is rightMovement.
  • animation-duration: Defines how long the animation will run, set to 5s.
  • animation-iteration-count: Defines how many times the animation will repeat. In this case, it is set to infinite for endless loops.
  • animation-timing-function: Controls the speed of the animation, here it is set to linear.
  • animation-direction: Controls the direction of the animation. The value alternate causes the animation to reverse direction after each cycle, making the box move back to its starting position before repeating the animation.

The @keyframes rule defines the animation behavior at different points in time. In this case, the box moves horizontally from left to right:

What happens in this example?

When the page is loaded, the box element will start from the left (top: 0, left: 0) and move horizontally to the right (top: 0, left: 1200px) over 5 seconds. After reaching the rightmost point, the box will reverse direction (due to animation-direction: alternate;) and move back to its original position, repeating this cycle infinitely.

Conclusion

This example demonstrates how the animation-direction: alternate; property can be used to make an animation move back and forth. By adjusting the properties like animation-duration, animation-timing-function, and @keyframes, you can create smooth, bidirectional animations that are visually engaging.

CSS Animation with Fill Mode

        width: 350px;

What is this example?

This example demonstrates the use of the animation-fill-mode: forwards; property to maintain the final state of an animation after it has finished running.

Explanation of Key Parts:

1. Global Reset

The following CSS resets the margin, padding, and box-sizing properties globally for all elements, ensuring consistent spacing and layout:


*{
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2. Container

The container takes up the full viewport width and height, with a beige background and black border:


.container{
    width: 100vw;
    height: 100vh;
    background-color: beige;
    border: 5px solid black;
}

3. Box Styling

The box element is styled with a blue background, yellow border, margin, and relative positioning, which allows the animation to affect its position:


.box{
    width: 350px;
    height: 80px;
    background-color: blue;
    border: 3px solid yellow;
    margin: 10px;
    position: relative;
}

4. Animation Properties

The animation is applied to the box element with these key properties:

  • animation-name: Specifies the name of the animation, which is rightMovement in this case.
  • animation-duration: Defines how long the animation runs, set to 5s.
  • animation-timing-function: Defines how the animation speeds up or slows down. It is set to linear here, meaning the animation maintains a constant speed.
  • animation-fill-mode: Specifies how the element's style should be affected before and after the animation. The value forwards makes the element retain the styles set at the end of the animation (i.e., the box will remain at its final position after the animation completes).

5. @keyframes Rule

The @keyframes rule defines the animation sequence, moving the box from the left to the right:


@keyframes rightMovement{
    from{
        top: 0;
        left: 0;
    }
    to{
        top: 0;
        left: 1200px;
    }
}

What happens in this example?

When the page is loaded, the box will start from its initial position (top: 0, left: 0) and move horizontally to the right (top: 0, left: 1200px) over 5 seconds. Once the animation completes, the animation-fill-mode: forwards; property ensures that the box stays in the final position (left: 1200px) instead of returning to its initial position.

Conclusion

This example illustrates the use of the animation-fill-mode: forwards; property, which ensures that an animated element retains its final state after the animation completes. This can be useful for creating animations where the element should remain in the end position, such as moving a box to a new location and keeping it there after the animation ends.
