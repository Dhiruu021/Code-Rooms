Definition and Examples

Different types of ordinary differential equations:

  • Definition: An equation involving a function and its derivatives.
  • Examples: dy/dx = x + y, y'' + y' - 6y = 0

Order and Degree

Understanding the order and degree of a differential equation:

  • Order: Highest derivative.
  • Degree: Power of the highest derivative.

Solutions of First Order Equations

Methods of solving first-order differential equations:

  • General Solution: Includes arbitrary constants.
  • Methods: Separation of variables, integrating factors.

Linear Differential Equations of First Order

First-order linear equations:

  • Bernoulli's Equation: Use substitution to solve non-linear equations.

Linear Differential Equations with Constant Coefficients

Linear differential equations with constant coefficients:

  • General Form: a_ny^(n) + a_(n-1)y^(n-1) + ... + a_1y' + a_0y = 0
  • Particular Integral: Solve using complementary function and particular integral.
