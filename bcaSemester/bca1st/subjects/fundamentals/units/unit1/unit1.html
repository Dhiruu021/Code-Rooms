Basic Components of a Computer System

Key components include:

  • Control Unit: Manages and coordinates system activities.
  • ALU (Arithmetic Logic Unit): Performs arithmetic and logical operations.
  • Input/Output: Handles communication with external devices.
  • Memory Unit: RAM and ROM , Hard drives, SSDs, and optical storage .
