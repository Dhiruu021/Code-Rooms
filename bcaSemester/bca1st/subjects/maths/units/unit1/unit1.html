Rank of a Matrix

Understanding the rank of a matrix:

  • Definition: The rank of a matrix is the maximum number of linearly independent rows or columns.
  • Methods to Find Rank: Using row-reduction techniques or determinant methods to find the rank.

Inverse of Matrix by Elementary Transformations

Methods of finding the inverse of a matrix using elementary transformations:

  • Elementary Row Operations: Using row swaps, scaling rows, and adding multiples of one row to another.
  • Row-Echelon Form: Transforming the matrix to row echelon form and then solving.
  • Gaussian Elimination: Applying this method to find the inverse of a matrix.

Consistency of Linear System of Equations

Understanding the consistency of linear systems:

  • Consistent System: A system of equations that has at least one solution.
  • Inconsistent System: A system of equations that has no solution.
  • Solving Methods: Using matrix methods such as Gauss elimination or Cramer's rule to determine the consistency.

Eigen Values and Eigen Vectors

Exploring eigenvalues and eigenvectors:

  • Eigenvalues: Scalar values that satisfy the equation Av = λv, where A is the matrix, λ is the eigenvalue, and v is the eigenvector.
  • Eigenvectors: Non-zero vectors that only change by a scalar factor when a linear transformation is applied.

Cayley-Hamilton Theorem

Understanding the Cayley-Hamilton theorem:

  • Theorem Statement: Every square matrix satisfies its own characteristic equation.
  • Application: Using the Cayley-Hamilton theorem to find powers of matrices and other operations.
