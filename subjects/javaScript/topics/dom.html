Text Manipulation

Properties and methods for manipulating text content within HTML elements:

  • innerText: Displays only the visible text within a node.
  • textContent: Shows all text, including hidden content.
  • innerHTML: Displays the full HTML markup within the node.

let para = document.querySelector("p");
console.dir(para.innerHTML);    // Shows extra tags like bold tags
console.dir(para.innerText);    // Shows only visible text on screen
console.dir(para.textContent);  // Shows hidden text as well

// Manipulation example
para.innerText = "Hi, I'm changing with the help of innerText";
para.innerHTML = "Hi, I'm changing with HTML tags";  // Recognizes HTML tags

Attribute Manipulation

Methods to access and modify attributes like class, id, and style:

  • getAttribute(attr): Get the current value of an attribute.
  • setAttribute(attr, val): Set or update the value of an attribute.

let box = document.querySelector("div");
box.getAttribute("id");  // Get the 'id' attribute
box.setAttribute("id", "newbox");  // Update 'id' to 'newbox'

Style Manipulation

Access and modify styles directly in JavaScript using camelCase naming conventions:


let aquaBox = document.querySelector("#box");
aquaBox.style.fontSize = "xxx-large";  // Change font size
aquaBox.style.color = "red";            // Change text color
aquaBox.style.backgroundColor = "grey"; // Change background color

Class Manipulation (classList)

Using classList to add, remove, or toggle classes on elements:

  • classList.add("className"): Adds a new class.
  • classList.remove("className"): Removes a class.
  • classList.toggle("className"): Toggles a class on or off.
  • classList.contains("className"): Checks if a class exists.

let obj = document.querySelector(".xyz");
obj.classList.add("NewClass");  // Adds a new class
obj.classList.toggle("abc");    // Toggles the 'abc' class

Navigate within the DOM Tree

Navigation in the DOM lets you move through elements based on their relationships. For example, finding an element's parent, child, or sibling without directly selecting them by ID or class.

Parent Element


// Selecting the parent element of an h1 tag
let heading1 = document.querySelector("h1");
let parent = heading1.parentElement; // Accessing the parent element of h1
console.dir(parent);
parent.style.backgroundColor = "#500763"; // Change background color of the parent

Child Elements

Child elements can be accessed as an HTML collection. Individual children are accessed by their index.


// Accessing child elements
let unorderList = document.querySelector(".items"); // Selecting parent element
let childrens = unorderList.children; // Getting children of the parent
console.dir(childrens);

Sibling Elements

Move to previous or next sibling elements in the DOM:


let unorderList2 = document.querySelector(".items");
let childrenAll = unorderList.children; 

// Accessing sibling elements
console.dir(childrenAll[1]); // Selecting specific child
console.dir(childrenAll[1].previousElementSibling); // Access previous sibling
console.dir(childrenAll[1].nextElementSibling); // Access next sibling
console.dir(childrenAll[1].nextElementSibling.nextElementSibling); // Access next of next sibling

Removing Elements in JavaScript

JavaScript provides methods like removeChild() and remove() to remove elements from the DOM.

Adding Elements for Removal Demo

In this example, we create new buttons and insert them around a paragraph element to demonstrate removal methods:


// Creating buttons to insert around a paragraph
let btn = document.createElement("button");
btn.innerHTML = "NEW BUTTON 1";
let btn2 = document.createElement("button");
btn2.innerHTML = "NEW BUTTON 2";
let btn3 = document.createElement("button");
btn3.innerHTML = "NEW BUTTON 3";
let btn4 = document.createElement("button");
btn4.innerHTML = "NEW BUTTON 4";

let para = document.querySelector(".para"); // Selects the 
 element
para.insertAdjacentElement('beforebegin', btn); // Adds btn before paragraph
para.insertAdjacentElement('afterbegin', btn2); // Adds btn2 as first child of para
para.insertAdjacentElement('beforeend', btn3);  // Adds btn3 as last child of para
para.insertAdjacentElement('afterend', btn4);   // Adds btn4 after paragraph

1. removeChild() Method

This method requires a parent element to remove a child element:


// Using removeChild() to remove btn (parent-child relationship)
let body = document.querySelector("body");
body.removeChild(btn); // Removes the btn created earlier

2. remove() Method

The remove() method can directly remove elements without specifying a parent:


// Using remove() to directly remove elements
btn2.remove(); // Removes btn2
para.remove(); // Removes the entire paragraph

Summary

  • removeChild() – Removes a specified child from a parent element.
  • remove() – Directly removes an element without specifying its parent.
