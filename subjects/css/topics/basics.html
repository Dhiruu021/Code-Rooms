Overview of CSS Basics

This section introduces internal CSS, which is defined in the <style> tag inside the HTML <head>. The use of the !important keyword allows internal CSS to override inline CSS, which normally has higher precedence.

CSS Selectors

CSS selectors specify which elements the CSS rules apply to:

  • Element Selector: Targets all elements of a specific type. For example, p selects all paragraphs.
  • ID Selector: Targets elements with a specific id attribute. IDs are unique per page.
  • Class Selector: Targets elements with a specific class. Multiple elements can share the same class.
  • Group Selector: Allows selection of multiple elements by separating selectors with a comma.

Example of Using Selectors

In the example code:

  • p, article { background-color: black; } - This sets a background color for both <p> and <article> elements.
  • p.class_name { background-color: black; } - This applies a background color only to paragraphs with the class class_name.
