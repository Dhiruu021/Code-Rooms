HTML Basics

Overview of HTML Basics

HTML (Hypertext Markup Language) is the foundational language used to create web pages. It provides a way to structure and present text, images, links, and other elements. Below, we cover essential HTML tags and their practical uses.

Basic Text Elements


<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Section Heading</h3>
<h4>Smaller Heading</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

<p>This is a paragraph, used for blocks of text.</p>

<h7>This tag is not valid and behaves like a paragraph.</h7>

Note: Only <h1> to <h6> are valid heading tags. Headings provide a hierarchical structure, with <h1> being the most important.

Links and Anchors


<a href="https://www.google.com">Go to Google</a>

<!-- Anchor (bookmark) link to jump within the same page -->
<a href="#bookmark">Go to Second Ordered List</a>

Links use the <a> tag with the href attribute to navigate to a URL. Adding #id in href lets you jump to a specific section within the page, known as a bookmark.

Lists


<ul>
    <li>Mango</li>
    <li>Papaya</li>
    <li>Grapes</li>
    <li>Guava</li>
</ul>

<!-- Unordered List: Useful for bullet points -->

<ol>
    <li>Step 1</li>
    <li>Step 2</li>
    <li>Step 3</li>
</ol>

<!-- Ordered List: Useful for steps or ordered items -->

Unordered lists (<ul>) display items with bullets, while ordered lists (<ol>) show them with numbers. Use unordered lists when item order doesnâ€™t matter, and ordered lists for steps or instructions.

Definition List


<dl>
    <dt>Coffee</dt>
    <dd>- Black hot drink</dd>
    <dt>Milk</dt>
    <dd>- White cold drink</dd>
</dl>

<!-- Definition lists (<dl>) are useful for listing terms and definitions -->

Definition lists (<dl>) display terms (<dt>) with their definitions (<dd>), ideal for glossary-like content.

Tables


<table>
    <tr>
        <td>Column 1</td>
        <td>Column 2</td>
        <td>Column 3</td>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
        <td>Row 1, Cell 3</td>
    </tr>
</table>

<!-- Tables are created with <table> tags, where rows (<tr>) contain cells (<td>) -->

Tables organize data into rows and columns. Use <tr> for rows and <td> for individual cells within a <table>.

Text Formatting


<strong>Bold (Strong Emphasis)</strong>
<b>Bold</b>
<i>Italic</i>
<em>Emphasis (Italic)</em>
<u>Underlined Text</u>
<s>Strikethrough</s>

<sup>Superscript</sup> and <sub>Subscript</sub>

For accessible text styling, use <strong> and <em> for bold and italic text, respectively. These tags also convey semantic meaning. <sup> and <sub> are useful for mathematical and chemical notations.

Image and Media Links


<img src="image.jpg" alt="Image description">

<a href="mailto:example@example.com">Email Us</a>
<a href="tel:+123456789">Call Us</a>

Use <img> to embed images, with src for the file path and alt to describe the image for accessibility. <a href="mailto:"> links email addresses, and tel: links phone numbers.

Preserving Text Formatting


<pre>
    Preformatted text
    maintains line breaks
    and spaces.
</pre>

<code>
    <!-- Code tag for inline code snippets -->
</code>

Use <pre> for text where whitespace and line breaks should be preserved, and <code> for inline code examples, especially useful for displaying code snippets.

Navigation Links


<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About Us</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>

The <nav> element is used to define navigation links, typically wrapped in an unordered list (<ul>) to group them.

Quotes


<q>This is a short quote.</q>

The <q> tag is used for inline quotes. For longer quotations, consider using <blockquote>.

Conclusion

This section provides a practical introduction to essential HTML elements for structuring and styling content on web pages. Using these elements appropriately enhances readability, accessibility, and page layout.

The HTML Head Tag and Meta Elements

Overview of the Head Tag

The <head> tag contains metadata and settings that help optimize a web page for search engines, compatibility, and user experience.

Document Type and Language

Starting with the <!DOCTYPE html> declaration, followed by the lang attribute, sets the document type and language for accessibility and search optimization.


<!DOCTYPE html>
<html lang="en">

Meta Tags

Meta tags in the <head> provide essential information for SEO, device responsiveness, and character encoding.

  • Character Encoding: meta charset="UTF-8" allows the use of multiple languages, such as Chinese or Arabic.
    • 
<meta charset="UTF-8">
  • Browser Compatibility: meta http-equiv="X-UA-Compatible" content="ie=edge" ensures the website renders consistently across different browser versions (typically uses the latest rendering mode).
    • 
<meta http-equiv="X-UA-Compatible" content="ie=edge">
  • Responsive Design: meta name="viewport" content="width=device-width, initial-scale=1.0" adjusts the page layout to fit various screen sizes and starts with an initial zoom of 100%.
    • 
<meta name="viewport" content="width=device-width, initial-scale=1.0">
  • Description: meta name="description" content="Learned Html With Love Babbar" helps search engines understand the content of the page for ranking.
    • 
<meta name="description" content="Learned Html With Love Babbar">

Title and Favicon

The <title> element defines the page title seen on the browser tab, while the link tag adds a favicon (icon next to the title).


<title>-Akhand</title>
<link rel="shortcut icon" href="/project_1/06_akhand.jpeg" type="image/x-icon">
  • Title: Visible in the browser tab; helps with SEO.
  • Favicon: A small icon associated with the website, enhancing brand recognition.
