Overview of Div Elements

The <div> tag is a block-level element in HTML. Unlike inline elements like <span>, block-level elements occupy the full width of their container and stack vertically. This makes <div> elements useful for creating structured layouts.

Example 1: Basic Div Styling

Setting background colors, padding, and borders on <div> elements is straightforward since they are block elements:


#box1 {
    width: 200px;
    height: 100px;
    background-color: lightblue;
    border: 2px solid black;
    padding: 10px;
}

#box2 {
    width: 200px;
    height: 100px;
    background-color: lightcoral;
    border: 2px dashed red;
    padding: 10px;
}

Result: Each <div> element appears as a block with specified background color, border, and padding.

Example 2: Changing Display to Inline and Inline-Block

To control layout, you can alter a <div> element’s display property:


/* Inline Display */
#box1 {
    display: inline;
    background-color: lightgreen;
    padding: 10px;
}

/* Inline-Block Display */
#box2 {
    display: inline-block;
    width: 150px;
    height: 50px;
    background-color: lightcoral;
    padding: 10px;
}

Result: With display: inline, #box1 takes up only the necessary width, placing it in line with adjacent elements. With display: inline-block, #box2 behaves inline but allows custom sizing.

This section demonstrates how <div> elements are versatile for structuring content, and how adjusting their display properties impacts layout.

box-sizing: border-box

Sample HTML Structure:

            
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
      * {
          padding: 0;
          margin: 0;
          box-sizing: border-box;
      }
    </style>
  </head>
  <body>
    <h1>Why <i>box-sizing: border-box;</i> is a Good Habit</h1>
    <p>
      WITHOUT IT, if we create a div of 300px*300px and add a 20px border, the box becomes 320px*320px.
      Increasing padding or margin also increases the box's total dimension.
    </p>
    <p>
      WITH IT, all padding and borders stay inside the defined dimensions, keeping the box size fixed.
      Using box-sizing: border-box universally applies this effect to all elements.
    </p>
  </body>
</html>

Overview of box-sizing: border-box

The box-sizing: border-box; property alters how elements calculate their width and height, keeping borders and padding within the defined size of the box. This is often used as a universal reset to create consistent layouts.

Example 1: Default Box Sizing Behavior (content-box)

By default, the box model follows box-sizing: content-box;, which means any added padding or border increases the total size of an element:


div {
    width: 300px;
    height: 300px;
    border: 20px solid black;
}

Result: The div now takes up 340px by 340px in total because of the added border.

Example 2: Using box-sizing: border-box for Consistent Sizing

Applying box-sizing: border-box; ensures that padding and borders are included within the element’s defined width and height:


* {
    box-sizing: border-box;
}

Result: Now, any padding or border is contained within the 300px by 300px box, maintaining consistent sizing.

This approach simplifies layouts and makes it easier to work with fixed dimensions, especially when dealing with padding and borders.
