What is a CSS Transition?

A CSS transition allows you to change the style of an element from one state to another over a specific duration. The transition starts when an event (like hovering or focusing) happens, and it moves smoothly from the initial state (source) to the final state (destination).

How is Transition Different from Animation?

Transitions and animations are both used to change an element's properties over time, but they differ in the following ways:

  • Animation: A combination of multiple transitions, where the element moves from one location to another and then continues moving to another position. It involves multiple keyframes (steps).
  • Transition: A one-step movement from the source to the destination state. It happens when an event is triggered and completes in a single motion.

In simple terms, transitions are a part of animations; animation uses multiple transitions to create complex movements.

Shorthand Notations for CSS Transitions

You can define transitions using shorthand notations, making your code more concise. Below are the common syntax rules:

1. Apply to 1 property:


/* property name | duration */
transition: margin-right 4s;

2. Apply to 1 property with delay:


/* property name | duration | delay */
transition: margin-right 4s 1s;

3. Apply to 1 property with easing function:


/* property name | duration | easing function */
transition: margin-right 4s ease-in-out;

4. Apply to 1 property with delay and easing:


/* property name | duration | easing function | delay */
transition: margin-right 4s ease-in-out 1s;

5. Apply to 1 property with behavior:


/* property name | duration | behavior */
transition: display 4s allow-discrete;

6. Apply to multiple properties:


/* Apply to 2 properties */
transition: margin-right 4s, color 1s;

7. Apply to all changed properties:


/* Apply to all changed properties */
transition: all 0.5s ease-out allow-discrete;
transition: 200ms linear 50ms;

8. Use global values:


/* Global values */
transition: inherit;
transition: initial;
transition: revert;
transition: revert-layer;
transition: unset;

Conclusion

CSS transitions are a great way to enhance user interaction with smooth visual effects. They are easy to implement and can be customized using the different shorthand properties to control duration, delay, easing, and the properties to be transitioned.

CSS Transition: Width Change

Sample HTML Structure:

            
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .box{
            height: 400px;
            width: 400px;
            background-color: rgb(255, 0, 0);
            border: 2px solid black;

            /* Using transition for width */
            transition-property: all;
            transition-duration: 2s;
            transition-delay: 1s;
            transition-timing-function: linear;
        }

        /* Hover effect */
        .box:hover{
            width: 1600px;
            background-color: azure;
        }
    </style>
</head>
<body>
    <div class="box">
        increasing width using transition.
    </div>
</body>
</html>

What is Happening in This Example?

In this example, we are demonstrating how to increase the width of an element using CSS transitions.

Key Parts of the Code:

  • Initial State: The .box element is a red square with a width of 400px.
  • Transition Properties: The transition-property is set to all, meaning all properties that change will be transitioned. In this case, width and background-color will be transitioned.
  • Transition Duration: The transition-duration is set to 2s, which means the transition will take 2 seconds to complete.
  • Delay: The transition starts after a 1-second delay (transition-delay: 1s).
  • Timing Function: The transition uses linear timing, which ensures the transition happens at a constant speed.

When Hovering Over the Box:

When the user hovers over the red box, the width increases from 400px to 1600px and the background color changes to azure. This change happens smoothly over 2 seconds, with a 1-second delay before it begins.

Transition Property Options:

There are multiple ways to define what properties should be transitioned. In this case:

  • transition-property: all; applies the transition to all properties that change.
  • To apply the transition only to the width, we could use transition-property: width;.

Conclusion:

Transitions allow smooth changes between styles. By specifying properties such as duration, delay, and timing function, you can control how and when the transition happens. In this case, we applied it to the width and background-color of the box.
