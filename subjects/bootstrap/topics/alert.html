Bootstrap Alerts

Overview

Alerts in Bootstrap provide a flexible way to display important messages to users. They can be used for various purposes, such as showing success messages, warnings, or error notifications. This section covers the different types of alerts and their usage.

Basic Alerts


<div class="alert alert-success" role="alert">
    A simple success alertâ€”Congratulations!
</div>

<div class="alert alert-danger" role="alert">
    Submission Failed - Please Try Again!
</div>

The examples above demonstrate basic success and danger alerts. Each alert has a contextual class that determines its style. Common classes include alert-success for success messages and alert-danger for error messages.

Dismissible Alerts

Alerts can be made dismissible, allowing users to close them easily. This can enhance user experience by reducing clutter. 


<div class="alert alert-warning alert-dismissible fade show" role="alert">
    <strong>Holy guacamole!</strong> Here this closing functionality has a Javascript Feature.
    <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>

The alert-dismissible class adds dismissible functionality. The btn-close button allows users to close the alert. The fade class provides a smooth transition when the alert is dismissed.

Alerts with Icons

Alerts can also include icons to convey meaning visually, improving clarity.


<div class="alert alert-success d-flex align-items-center" role="alert">
    <svg class="bi flex-shrink-0 me-2" role="img" aria-label="Success:"><use xlink:href="#check-circle-fill"/></svg>
    <div>
      An example success alert with an icon
    </div>
</div>

The alert above shows an icon next to the alert message. The use of SVG icons enhances the visual representation of the alert's purpose.

Conclusion
