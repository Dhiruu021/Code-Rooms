Motherboard

Introduction: The main circuit board of a computer.
Components: Includes CPU socket, RAM slots, chipset, and expansion slots.
Types: ATX, Micro-ATX, ITX.
Form Factor: Defines the physical dimensions and layout of the motherboard.

Memory: RAM, ROM, SRAM, DRAM

Details of computer memory types:

  • RAM: Temporary storage for data being processed.
  • ROM: Read-only memory for permanent data.
  • SRAM: Static RAM, faster and used for cache memory.
  • DRAM: Dynamic RAM, used as main system memory.

Buses

Introduction: Communication pathways for data transfer.
Types:

  • Data Bus
  • Address Bus
  • Control Bus
Utility: Facilitates data exchange between components.

Add-on Cards

Expansion cards that enhance the functionality of a computer, such as graphics cards and network cards.
