Input Devices

Devices that allow data to be input into a computer system:

  • Keyboard
  • Mouse
  • Digitalizing Tablet
  • Scanners
  • Bar-Code Reader
  • Light Pen
  • Touch Screen

Output Devices

Devices that display or output information from a computer system:

  • Monitors: Characteristics and types (Digital, Analog, Size, Resolution, Refresh Rate, Video Standards).
  • Types of Printers
  • Plotters
  • Sound Card and Speakers

Storage Devices

Data storage and retrieval methods categorized as primary vs secondary storage:

  • Primary Storage: RAM and ROM.
  • Secondary Storage: Various devices such as:
    • Magnetic Tape
    • Magnetic Disks
    • Cartridge Tape
    • Data Drives
    • Hard Disk Drives
    • Floppy Disk
    • Optical Disk
