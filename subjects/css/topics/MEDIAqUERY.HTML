Overview of Media Queries

Media queries in CSS are used to apply styles based on the characteristics of the device or viewport, such as its width, height, or orientation. This allows developers to create responsive web designs that adapt to different screen sizes and resolutions.

Syntax of Media Queries

The general syntax for a media query looks like this:


@media (condition) {
    element {
        /* Apply styles */
    }
}

For example, you can apply different styles when the viewport width is less than 500px:


@media (max-width: 500px) {
    .box {
        background-color: blue;
    }
}

Example: Changing Background Color Based on Screen Width

In this example, the background color of the box will change depending on the screen width:

            
@media (min-width: 300px) and (max-width: 500px) {
    .box {
        background-color: blue;
    }
}

Here, when the screen width is between 300px and 500px, the box background changes to blue. If the screen is larger than 500px, it defaults to the original color (aqua in the previous example).

Other Media Query Examples

  • @media (min-width: 500px): Applies styles when the viewport is at least 500px wide.
  • @media screen and (min-width: 500px): Targets screens that are at least 500px wide, specifically on screen devices (not for print, etc.).
  • @media (max-height: 600px): Applies styles when the viewport height is 600px or less.

Summary of Media Queries

Media queries are essential for building responsive web designs. By targeting different screen sizes and devices, you can ensure that your website looks great and functions properly on all devices, from mobile phones to large desktop screens.
