Overview of CSS Shadow

CSS provides two types of shadow effects that can be applied to elements: box-shadow and text-shadow. These shadows enhance the visual presentation by adding depth and emphasis to elements on a page.

Types of Shadow in CSS

  • Box Shadow: Adds shadow effects around an element's frame.
  • Text Shadow: Adds shadow effects to text.

Box Shadow Syntax

The box-shadow property is used to apply shadow effects to an element. It takes up to five values:


box-shadow: 10px 5px 5px black; /* Horizontal offset, vertical offset, blur radius, and color */

Examples of Box Shadow

1. Basic Box Shadow:


box-shadow: 10px 5px 5px black;

This adds a simple shadow with a 10px horizontal and 5px vertical offset, with a 5px blur radius and a black color.

2. Multiple Shadows:


box-shadow: 3px 3px red inset, -1em 0 0.4em olive;

This adds two shadows: a red inset shadow and an olive-colored shadow on the outer edge.

3. Inset Box Shadow:


box-shadow: inset 5em 1em gold;

The inset keyword changes the shadow from an outer shadow to an inner shadow.

Text Shadow Syntax

The text-shadow property is used to apply shadow effects to text. It takes up to four values:


text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5); /* Horizontal offset, vertical offset, blur radius, and color */

Global Values for Box Shadow

Box shadows can also use global values to control inheritance and resetting:


box-shadow: inherit; /* Inherits the value from the parent element */
box-shadow: initial; /* Resets to the default value */
box-shadow: unset; /* Removes any applied shadow */

Summary of CSS Shadow

Shadows in CSS add depth to elements, enhancing the design of your web page. Use box-shadow for element shadows and text-shadow for text effects. By customizing values like color, blur radius, and offsets, you can achieve various visual effects for modern web designs.

Overview of CSS Box Shadow

In CSS, the box-shadow property is used to apply shadow effects around an element's box. It is particularly useful for giving depth to an element by making it appear as though it is floating above the page. The box-shadow can be customized with multiple values for horizontal and vertical offsets, blur radius, spread radius, and color.

Box Shadow Syntax

The basic syntax for the box-shadow property is:


box-shadow: h-offset v-offset blur spread color inset;
  • h-offset: Horizontal shadow offset (positive for right, negative for left).
  • v-offset: Vertical shadow offset (positive for down, negative for up).
  • blur: The blur radius (increases shadow softness).
  • spread: The spread radius (increases or decreases shadow size).
  • color: The color of the shadow.
  • inset (optional): Makes the shadow appear inside the element.

Example of Box Shadow

The following example applies a shadow effect with a horizontal offset of 10px, a vertical offset of -60px, and a brown color to the box:


box-shadow: 10px -60px brown;

In this case, the shadow is positioned 10px to the right and 60px above the element.

Changing the Shadow's Position

By changing the horizontal and vertical offsets, you can control the shadow's direction:


box-shadow: -10px 10px gray; /* Shadow is 10px left and 10px down */

Notes on Shadow Offsets:

- Positive values for the horizontal offset move the shadow to the right, and negative values move it to the left.
- Positive values for the vertical offset move the shadow down, and negative values move it up.

Shadow Blur

The blur radius controls the softness of the shadow. A larger value creates a softer, more diffuse shadow:


box-shadow: 10px 10px 15px gray; /* Larger blur radius creates a softer shadow */

Inset Shadow

Adding the inset keyword creates an inner shadow effect:


box-shadow: inset 5px 5px 10px #000000; /* Creates a shadow inside the element */

Global Values for Box Shadow

The box-shadow property can also use global values to control inheritance or resetting:


box-shadow: inherit; /* Inherits the value from the parent element */
box-shadow: initial; /* Resets to the default value */
box-shadow: unset; /* Removes any applied shadow */

Summary of Box Shadow

Box shadows in CSS help enhance the visual presentation of elements by making them appear to float above the page. You can control the shadow's position, softness, spread, and color to create unique visual effects that add depth to your design.

Overview of CSS Box Shadow with Blur

The box-shadow property in CSS allows you to create a shadow effect around an element. By adding a blur radius, you can make the shadow appear softer or more diffuse. The blur value controls the spread of the shadow, making it less sharp and more spread out the higher the value.

Box Shadow Syntax with Blur

For a shadow effect that includes a blur radius, the syntax looks like this:


box-shadow: h-offset v-offset blur color;
  • h-offset: The horizontal offset of the shadow (positive for right, negative for left).
  • v-offset: The vertical offset of the shadow (positive for down, negative for up).
  • blur: The blur radius which controls the softness of the shadow.
  • color: The color of the shadow.

Example of Box Shadow with Blur

In the following example, the box shadow has a horizontal offset of 10px, a vertical offset of 60px, and a blur radius of 50px:


box-shadow: 10px 60px 50px brown;

This creates a shadow that is 10px to the right, 60px down, and has a blur radius of 50px, resulting in a soft shadow effect.

Changing the Shadow's Blur

The blur radius controls the sharpness of the shadow. A higher blur value makes the shadow more diffuse:


box-shadow: 10px 60px 100px brown; /* Larger blur radius creates a softer shadow */

Shadow Offset with Blur

You can adjust both the shadow's offset and the blur to achieve different visual effects. A combination of larger offset and blur results in a more distant and softer shadow:


box-shadow: -20px 30px 60px rgba(0, 0, 0, 0.5); /* Shadow moves to the left and down, with a blur effect */

Notes on Blur:

- A larger blur value will make the shadow spread out, creating a more subtle effect.
- A smaller blur value makes the shadow sharper and more defined.

Global Values for Box Shadow

The box-shadow property can use global values to inherit or reset shadow styles:


box-shadow: inherit; /* Inherits the shadow from the parent element */
box-shadow: initial; /* Resets to the default value */
box-shadow: unset; /* Removes any applied shadow */

Summary of Box Shadow with Blur

The box-shadow property with blur radius is a powerful tool for creating shadows in CSS. By adjusting the horizontal and vertical offsets as well as the blur radius, you can control the appearance of the shadow to suit your design needs. The blur radius specifically helps to soften the shadow and make it more visually appealing.

Overview of CSS Box Shadow with Spread Radius

The box-shadow property in CSS allows you to add a shadow effect around an element. The spread radius determines how much the shadow spreads beyond the element. A positive value increases the size of the shadow, while a negative value makes it shrink. The spread radius is applied after the blur radius.

Box Shadow Syntax with Spread

To create a box shadow with a spread radius, you can use the following syntax:


box-shadow: h-offset v-offset blur spread color;
  • h-offset: The horizontal offset of the shadow (positive for right, negative for left).
  • v-offset: The vertical offset of the shadow (positive for down, negative for up).
  • blur: The blur radius which controls the softness of the shadow.
  • spread: The spread radius, which determines the size of the shadow.
  • color: The color of the shadow.

Example of Box Shadow with Spread Radius

In the following example, the box shadow has a horizontal offset of 10px, vertical offset of 30px, a blur radius of 10px, and a spread radius of 40px:


box-shadow: 10px 30px 10px 40px brown;

This creates a shadow that is 10px to the right, 30px down, with a blur radius of 10px, and the shadow spreads outward by 40px, making the shadow appear larger around the element.

Changing the Shadow Spread

The spread radius controls how much the shadow grows outward. A positive spread value increases the shadow's size, while a negative value shrinks it:


box-shadow: 10px 30px 10px 80px brown; /* Larger spread radius */

In this case, the shadow will spread much farther from the element, creating a larger and more diffused effect.

Example of a Negative Spread Radius

Using a negative spread radius will shrink the shadow, pulling it closer to the element:


box-shadow: 10px 30px 10px -20px brown; /* Negative spread radius */

This will create a shadow that appears tighter and smaller around the element.

Notes on Spread Radius:

- A positive value increases the size of the shadow, making it appear more distant.
- A negative value pulls the shadow in closer to the element, making it tighter.

Global Values for Box Shadow

As with other properties, the box-shadow property can use global values to inherit or reset shadow styles:


box-shadow: inherit; /* Inherits the shadow from the parent element */
box-shadow: initial; /* Resets to the default value */
box-shadow: unset; /* Removes any applied shadow */

Summary of Box Shadow with Spread Radius

The box-shadow property with a spread radius allows you to control the size of the shadow around an element. By adjusting the horizontal, vertical offsets, blur, and spread radius, you can create unique shadow effects that enhance the design of your webpage. The spread radius is especially useful when you want to control how large or small the shadow appears.

Overview of CSS Box Shadow with Inset

The box-shadow property in CSS can be used to create an inset shadow effect inside an element. This effect gives the illusion that the content is pressed into the background, as opposed to casting a shadow outside the element.

Box Shadow Syntax with Inset

To create an inset shadow, simply add the inset keyword before the shadow's properties:


box-shadow: inset h-offset v-offset blur spread color;
  • h-offset: The horizontal offset of the shadow (positive for right, negative for left).
  • v-offset: The vertical offset of the shadow (positive for down, negative for up).
  • blur: The blur radius which controls the softness of the shadow.
  • spread: The spread radius, which determines the size of the shadow.
  • color: The color of the shadow.

Example of Box Shadow with Inset

In the following example, an inset shadow is applied to the box. The shadow is created with a horizontal offset of 10px, vertical offset of -15px, and a brown color:


box-shadow: inset 10px -15px brown;

This creates a shadow effect that is applied inside the element, giving it a "pressed" look.

Effect of Inset Shadows:

  • Inset shadows make the element appear as if it's recessed into the page.
  • They provide a sense of depth and are great for creating button-like effects or card designs.

Changing the Inset Shadow

You can adjust the offsets, blur, and spread values to change the appearance of the inset shadow. Here is an example with a larger blur and spread:


box-shadow: inset 15px 30px 20px 5px brown;

This creates a more prominent inset shadow with a wider spread and softer edges.

Notes on Inset Shadows:

- The inset keyword is what differentiates an inside shadow from the default outside shadow.
- Inset shadows are commonly used for designs such as buttons, input fields, or containers that appear to be indented.

Global Values for Box Shadow

As with other properties, the box-shadow property can use global values to inherit or reset shadow styles:


box-shadow: inherit; /* Inherits the shadow from the parent element */
box-shadow: initial; /* Resets to the default value */
box-shadow: unset; /* Removes any applied shadow */

Summary of Box Shadow with Inset

The box-shadow property with the inset keyword creates a shadow inside the element, making it appear indented. By adjusting the shadow's offset, blur, and spread values, you can create unique visual effects, making your designs more dynamic and engaging.

Overview of CSS Multiple Box Shadows

The box-shadow property can accept multiple shadow values, separated by commas. This allows you to apply more than one shadow effect to an element, creating depth, layering, and multiple lighting effects in a single design.

Multiple Box Shadow Syntax

When using multiple shadows, you define each shadow's properties (offset, blur, spread, color) separated by commas. The shadows will be applied in the order they are written, one on top of the other:


box-shadow: h-offset1 v-offset1 blur1 spread1 color1, h-offset2 v-offset2 blur2 spread2 color2;
  • h-offset: The horizontal offset of the shadow.
  • v-offset: The vertical offset of the shadow.
  • blur: The blur radius, which controls how soft the shadow edges are.
  • spread: The spread radius, controlling how much the shadow grows.
  • color: The color of the shadow.

Example of Multiple Box Shadows

In the following example, two shadows are applied to the element:


box-shadow: 10px 15px brown, 15px 20px green;

The first shadow has a 10px horizontal offset, 15px vertical offset, and brown color. The second shadow has a 15px horizontal offset, 20px vertical offset, and green color.

Effect of Multiple Shadows:

  • Each shadow is applied one after the other, with the first shadow on top of the second.
  • The shadows are stacked, allowing for complex and creative effects, such as a shadow on shadow or layered highlights.

Changing Multiple Box Shadows

You can modify the position, size, and color of each shadow individually. Here’s an example with different shadow colors and blur values:


box-shadow: 10px 15px 10px brown, 15px 20px 20px rgba(0, 255, 0, 0.5);

The first shadow has no blur, while the second shadow has a blur radius of 20px, and the color is partially transparent green.

Layering Shadows

By adjusting the offsets and blur radii, you can create layered shadow effects:


box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.4), 15px 15px 15px rgba(0, 0, 0, 0.2);

This creates a layered effect where the first shadow is a soft, large blur and the second shadow is sharper and smaller.

Notes on Multiple Shadows:

  • Shadows are drawn in the order they are listed, with the first shadow being the closest and the last shadow being the farthest.
  • You can create interesting effects by combining hard and soft shadows, or using semi-transparent shadows for more subtle effects.

Summary of Multiple Box Shadows

Using multiple shadows in the box-shadow property allows you to create complex, multi-layered effects that can enhance your design. Experimenting with different offsets, blur, spread, and color values will help you achieve unique and dynamic visual results.

Overview of CSS Text Shadows

The text-shadow property allows you to add shadow effects to text. It is defined using the following syntax:


text-shadow: h-offset v-offset blur-radius spread-radius color;
  • h-offset: Horizontal offset of the shadow (required).
  • v-offset: Vertical offset of the shadow (required).
  • blur-radius: The blur radius of the shadow (optional, default is 0).
  • spread-radius: The spread radius of the shadow (optional, default is 0).
  • color: The color of the shadow (optional, default is black).

Examples of Text Shadows

Method 1: Basic Shadow


text-shadow: 2px 3px 2px white;

This example creates a white shadow with a 2px horizontal offset, a 3px vertical offset, and a 2px blur radius.

Method 2: Colored Shadow


text-shadow: yellow 2px 3px 2px;

This example adds a yellow shadow with a 2px horizontal offset, a 3px vertical offset, and a 2px blur radius.

Method 3: Shadow Without Blur


text-shadow: yellow 2px 3px;

This creates a sharp yellow shadow without any blur, just an offset in both horizontal and vertical directions.

Method 4: Default Shadow


text-shadow: 2px 3px;

If you don't specify a color, the default color is black, and you will see a simple shadow with the specified offset.

Method 5: Multiple Shadows


text-shadow: yellow 2px 3px 5px, brown 3px 4px;

Multiple shadows can be added by separating them with commas. In this case, a yellow shadow with a blur radius and a brown shadow with an offset are applied.

Practical Uses of Text Shadows

Text shadows are commonly used for:

  • Creating a 3D effect for text.
  • Making text stand out against a background.
  • Improving readability on complex backgrounds.
  • Creating glowing or neon effects for text.

Notes on Text Shadows

  • Using too many text shadows can make the text hard to read. It's best to use shadows sparingly and ensure they complement the design.
  • You can create "glowing" text effects by using bright colors and a higher blur radius.

Summary of Text Shadows

The text-shadow property is a powerful tool for enhancing text visibility and creating stylistic effects. By adjusting the offsets, blur radius, and color of the shadow, you can achieve various design aesthetics, such as glowing text or a 3D-like effect.
